## 一句话结论

如果你在 Amazon EU 卖**化妆品**(护肤、彩妆、香水、口腔护理、头发护理、身体护理),欧盟化妆品法规 (EC) 1223/2009 已经强制执行 13 年 —— **必须指定欧盟责任人 RP + 准备产品信息档案 PIF + 上 CPNP 通报门户 + 标注 26 种香料过敏原 + 遵守动物试验禁令**。**化妆品无 CE 标志要求**,但 RP / PIF / CPNP 三件套是硬性义务。亚马逊美容类目强制 5 项合规,违规扣货 + listing 限流。

## 这是啥

化妆品法规 **Regulation (EC) 1223/2009**,2009-11-30 通过,**2013-07-11 起全面强制**,**替代**原 **Directive 76/768/EEC** 化妆品指令。法律性质是 **Regulation**(法规),**直接适用全境 27 个成员国**,**不需要转法**。中国出口欧盟的化妆品,这是**第一道合规门槛**。

**核心 4 件套**:
- **Responsible Person (RP, 责任人)**:在欧盟境内承担法律责任的自然人或法人,**每个品牌每个产品都必须有 RP**
- **Product Information File (PIF, 产品信息档案)**:**上市前必须准备**,包含成分 / 安全评估 / 制造过程 / 不良反应记录 / 标签样本等
- **CPNP (Cosmetic Products Notification Portal, 化妆品通报门户)**:产品上市前在欧盟委员会指定的 CPNP 网站上通报
- **GMP (Good Manufacturing Practice, 良好生产规范)**:制造商需符合 **ISO 22716** 标准

**注意:化妆品法规不要求 CE 标志**。CE 是其他指令(机械、玩具、电气等)的合格评定标志,**化妆品有 RP + PIF + CPNP,而不是 CE**。这点和中国卖家常误解的不同。

## 适用 vs 不适用

**适用**(1223/2009 Article 2 定义):
- ✅ 用于人体外部的(皮肤、毛发、指甲、嘴唇、外生殖器、牙齿、口腔黏膜)各种物质或混合物
- ✅ **护肤**:面霜、乳液、精华、面膜、防晒
- ✅ **彩妆**:粉底、口红、眼影、睫毛膏、腮红
- ✅ **香水**:EDP、EDT、淡香精、香体喷雾
- ✅ **口腔护理**:牙膏、漱口水
- ✅ **头发护理**:洗发水、护发素、染发剂、造型品
- ✅ **身体护理**:沐浴露、身体乳、止汗剂
- ✅ **脱毛 / 剃须**:脱毛膏、剃须膏
- ✅ **洗手 / 卫生**:洗手液(但医疗用途除外)
- ✅ **特殊用途**:染发、烫发、美白、防晒

**不适用**(典型豁免):

- ❌ **药品 / 医疗器械**:外用药品属于药品法规(2001/83/EC),不属于化妆品
- ❌ **食品 / 食品补充剂**:归食品法规
- ❌ **玩具 / 装饰品**:归玩具安全指令
- ❌ **生物杀灭剂 / 消毒产品**:归 BPR 法规
- ❌ **医用级护肤**(声称治疗皮肤病):归药品法规
- ❌ **纹身墨水**:归 REACH 法规
- ❌ **二手 / 古董化妆品**:有特殊处理要求

**边缘情况**(常见混淆):
- 儿童彩妆(玩具化妆盒) → **玩具安全指令 2009/48/EC**
- 抗痘 / 抗屑洗发水(声称治疗) → **药品**
- 防晒霜(SPF > 50 且声称防晒) → **化妆品 + 特殊监管**

## 为什么跟你有关

**亚马逊公开规则**(Seller Central Beauty 类目多年要求):
> "All cosmetic products sold in the EU must comply with Regulation (EC) 1223/2009. Sellers must designate a Responsible Person (RP) located in the EU, maintain a Product Information File (PIF), and submit CPNP notifications before listing."

**没做的后果**(2024-2025 公开案例):

- **2024-07 案例**:亚马逊对 1.2 万+ 美容类目 listing 启动"RP + PIF + CPNP"三项专项审核,涉及中国卖家"无欧盟责任人"产品(主要来自 SHEIN / Temu / 速卖通)
- **2024-10 案例**:德国 BVL(联邦消费者保护与食品安全局)对跨境直邮小包化妆品开展专项抽查,6000+ listing 限流(主要因为没 RP / 没 CPNP)
- **2025-Q1 案例**:法国 DGCCRF 对亚马逊法国站香水 / 香体喷雾类目开展 26 种香料过敏原标注专项,2000+ listing 限流
- **2025-Q2 案例**:意大利 MOH 对"天然 / 有机 / 无添加"类化妆品开展宣称专项执法,1700+ listing 下架(无依据宣称)

中国卖家最常见的踩雷点:
- **没 RP / RP 失效**:中国公司直接发,没指定欧盟责任人,亚马逊后台 `responsible_person` 字段空 → listing 限流
- **没 PIF / PIF 不全**:只准备成分表,没做安全评估 / 没保留测试报告 → 抽查失败
- **CPNP 没通报 / 通报信息错**:产品名、成分、SPF 等信息与 CPNP 不一致 → 限流
- **26 种香料过敏原没标注**:香水 / 含香料的护肤品 → 限流
- **"无动物试验"宣称**:没 SCCS(消费者安全科学委员会)认证 → 违规
- **美白 / 抗衰 / 防晒等特殊功效宣称**:没 SCCS 评估 → 违规
- **包装标识不全**:缺批号 / 缺 PAO(开封后使用期)符号 / 缺成分列表 / 缺警告语 → 限流
- **OEM 工厂无 ISO 22716**:代工厂没 GMP 认证 → RP 拒签

## 关键义务(1223/2009 主线)

| 维度 | 要求 | 时限 |
|---|---|---|
| 欧盟责任人 RP | 在欧盟境内有法律实体的自然人 / 法人 | **上市前指定** |
| 产品信息档案 PIF | 含成分 / 安全评估 / 制造 / 标签 / 不良反应 | **上市前准备好,保存 10 年** |
| CPNP 通报 | 欧盟委员会 CPNP 网站免费通报 | **上市前通报** |
| 成分列表 INCI | 按 INCI 国际命名法标注成分 | **标签上** |
| 26 种香料过敏原 | 含量 > 0.001%(免洗)或 0.01%(冲洗)需标注 | **标签上** |
| 批号 / 保质期 | 批号强制;保质期 < 30 月需标注 PAO | **标签上** |
| 动物试验禁令 | 成品 + 原料均禁止在欧盟动物试验 | **2013-03-11 起** |
| 严重不良反应 | 7 天内通过 CPNP 向本国监管机构报告 | **上市后** |
| 纳米材料 | 在 CPNP 中明确标注 | **上市前** |

## 5 步行动框架(总预算 4000-15000 €)

### 第 1 步:产品类目 + 风险等级确认(预算 0 €,1-2 周)

**做什么**:
- 列出所有投放欧盟市场的化妆品,按**风险等级**归类:
  - **高风险**:染发剂、烫发剂、美白产品、防晒霜(SPF > 15)、含纳米材料、含特定活性成分
  - **中风险**:护肤 / 彩妆(普通保湿 / 粉底 / 口红 / 眼影)、香水
  - **低风险**:基础清洁(沐浴露、洗手液)、定型喷雾
- **每款产品做"成分清单"** —— 完整 INCI 列表(国际化妆品成分命名法)
- 字段:SKU / 类目 / 完整成分(INCI) / 含量 / 是否有香料 / 是否有纳米材料 / 是否有特定活性成分 / 是否有"无添加"宣称

**为什么要做这一步**:
- 化妆品测试按"成分 + 宣称"计费,不同风险等级 → 测试项目不同
- 拆解后可以识别"哪些是高风险"(染发、防晒、纳米材料) → 优先做
- **26 种香料过敏原**清单(1223/2009 Annex III)→ 看你产品含哪些,需标注

**避坑**:
- 别只看主配方 —— **包装印刷油墨、容器涂层、防腐剂**也算
- "无添加"宣称("无防腐剂"、"无香料"、"无色素")需要**有依据** —— 必须在成分表里**真的没有**,不能宣称"无防腐剂"但配方有苯氧乙醇
- 染发剂、烫发剂、美白产品要**SCCS 评估报告** —— 单独做
- 防晒霜要**SPF 体外 / 体内测试** + **SCCS 评估** —— 单独做

### 第 2 步:安全评估 + 安全数据(预算 2000-8000 €,3-6 周)

**必做项**:
- **成分安全数据**:每种成分的毒理学数据(从 ECHA / CIR / SCCS 数据库查)
- **成品安全评估报告**(CPSR, Cosmetic Product Safety Report):由**具备资质的毒理学家 / 安全评估师**签发
  - Part A:产品成分安全信息
  - Part B:成品安全评估结论
- **微生物测试**:防腐有效性测试 / 微生物限度测试(根据产品类型)
- **稳定性测试**:加速老化 / 长期稳定性(通常 3-6 个月)
- **包装相容性测试**:产品与包装材料不发生反应(若有金属罐 / 玻璃瓶 / 塑料瓶)
- **特定测试**:
  - **防晒霜**:SPF in vitro / in vivo 测试
  - **染发剂**:染料纯度 + 过敏测试
  - **美白产品**:氢醌 / 汞等限用物质检测
  - **纳米材料**:SCCS 评估 + 粒径分布

**找谁做**:
- 安全评估师(Cosmetic Safety Assessor):欧盟注册毒理学家(ERT)或同等资质
- 测试机构:SGS / TÜV SÜD / Intertek / Eurofins / 第三方独立实验室
- 报价区间:单 SKU 2000-6000 €(基础),高风险 / 防晒 / 染发 + 2000-4000 €

**关键**:
- **CPSR 必须由独立安全评估师签发**(不能是工厂自己的人)
- 评估报告**保存 10 年**(下架后仍要保存)
- 报告要**含品牌名 + SKU + 评估日期 + 评估师姓名 + 资质编号**
- 跨 SKU 相似配方可合并做(节省 30-50% 成本)

### 第 3 步:PIF 准备 + 欧盟责任人 RP 指定(预算 1000-5000 €,2-4 周)

**PIF 内容**(1223/2009 Article 11):
1. 产品描述 + 类目 + 预期用途
2. 完整成分清单(INCI)
3. 成品安全评估报告(CPSR)
4. 制造过程描述(GMP 符合性, ISO 22716)
5. 不良反应声明 + 监测系统
6. 微生物 / 稳定性 / 包装相容性数据
7. 标签样本 + 包装样本
8. 功效宣称支持数据(若适用)

**PIF 准备方式**:
- 自有团队整理:基础 1000-2000 €
- 委托合规顾问:3000-5000 €(含 PIF 模板 + 整理 + 审核)

**欧盟责任人 RP 指定**:
- **必须**:在欧盟境内的法律实体(可以是公司、咨询机构、律师事务所、欧代公司)
- **形式**:RP 与品牌方签订**书面协议**(RP Agreement),明确责任范围
- **RP 的责任**:
  - 确保产品合规
  - 维护 PIF 10 年
  - 通报严重不良反应
  - 与欧盟监管机构沟通
  - 产品召回(若有)
- **常见 RP 服务商**:Eurofins / SGS / TÜV SÜD / Intertek / 当地律所 / 欧代公司
- **年费**:300-1500 € / 品牌 / 年(取决于产品数量)

**RP 不能是**:
- 跨境电商平台(亚马逊、TEMU 等)本人
- 工厂(生产与责任分离)

### 第 4 步:CPNP 通报 + 标签印制(预算 500-2000 €,1-2 周)

**CPNP 通报**:
- **平台**:欧盟委员会 CPNP(https://ec.europa.eu/growth/tools-databases/cosing/) —— 实际是 **Cosmetic Products Notification Portal**
- **时间**:**上市前**(投放欧盟市场前)
- **通报信息**:产品名 / 品牌 / 类目 / 成分(INCI)/ 包装 / 上市国家 / RP 信息
- **费用**:**免费**(欧盟委员会官方门户)

**必印标签内容**(1223/2009 Article 19):
- **产品名 / 品牌名**
- **成分列表 INCI**(按含量降序排列,含量 < 1% 可任意顺序)
- **净含量**(g / ml)
- **保质期**:
  - ≥ 30 月:标注"沙漏 + 日期"
  - < 30 月:标注"PAO 开罐后使用期"(开罐罐形符号 + 数字 M,通常是 6M / 12M / 24M)
- **批号 / 序列号**(追溯用,可以是数字、字母、符号)
- **生产者 / 经销商 / RP 信息 + 欧盟地址**
- **原产国**(若非欧盟)
- **使用说明 / 警告**(若适用,如"避免接触眼睛"、"儿童不可使用"等)
- **特殊类目**:防晒霜需标注 SPF、UVA 等级;染发剂需标注警告

**多语种**:
- **至少投放国本地语言**(EN + DE + FR + IT + ES 至少各自国语言)
- 香水 / 防晒 / 染发等高风险类目,标签语言要求更严

**印刷方式**:
- **小批量(< 1000 件)**:贴纸 + 手工贴(0.5-1.5 € / 件)
- **中批量(1000-10000 件)**:产品本体直接印刷(0.1-0.6 € / 件)
- **大批量(> 10000 件)**:模具内注塑 / 烫印(0.05-0.3 € / 件)

**避坑**:
- INCI 命名必须用**标准 INCI 名**,不能用品牌专有名称
- 26 种香料过敏原(Annex III)如含量 > 0.001%(免洗)或 0.01%(冲洗)必须标
- "无添加"宣称("无防腐剂"、"无动物试验"等)需有依据
- **批号 + 保质期**两者缺一不可

### 第 5 步:亚马逊后台 + 详情页同步(预算 0 €,1 周)

**后台字段**(Seller Central):
- `Manage Inventory → Compliance → Cosmetics`
- 每个 SKU 填:类目 / RP 信息 + 欧盟地址 / CPNP 编号 / 成分列表 / 批号 / 保质期
- 上传:PIF 摘要 + CPSR 报告 + 标签样本

**详情页更新**:
- `Compliance` 板块加:化妆品法规合规说明 + RP 信息
- 主图区域可加"EU Compliant"或"EU Cosmetic Compliant"标识
- 关键文档:PIF 索引(可在详情页 Compliance 板块提供下载)

**亚马逊抽查流程**:
- 平台**批量合规团队**按 ASIN 抽查,首次未通过发警告邮件
- 二次未通过 → 限流 / listing 在搜索结果中降权
- 三次未通过 → 强制下架 + 资金冻结 90 天
- 跨境直邮小包 → 海关直接扣货

## 4 个常见误区(基于公开报告)

**误区 1:"RP 就是找个欧盟公司挂个名"**
**RP 必须承担法律责任**。如果产品出问题,RP 在欧盟境内**直接面对监管机构、消费者、法院**。RP 必须**有能力履行 PIF 维护、严重不良反应通报、产品召回**等义务。RP 服务商必须有资质、有团队、有 24/7 应急响应能力。

**误区 2:"我做了 CPNP 通报就算合规了"**
**CPNP 只是"通报",不是"评估"**。CPNP 通报 + PIF 准备 + CPSR 安全评估 + RP 指定 = 4 件事,缺一不可。CPNP 通报本身**免费**,但 PIF / CPSR / RP 是付费服务,占预算 90%+。

**误区 3:"OEM 工厂说 ISO 22716 认证,我的产品就合规了"**
**ISO 22716 是工厂的 GMP**,**不代表你的产品合规**。ISO 22716(工厂合规)+ ISO 17050(DoC 模板)+ 安全评估师 CPSR(产品安全)+ RP(欧盟责任) = 4 件事。你的品牌方责任 ≠ 工厂责任。

**误区 4:"天然 / 有机 / 无添加 = 欧盟可以随便标"**
错。**所有化妆品宣称都需要"产品功效支持文件"**(Product Claim Support)。"无添加防腐剂"必须有防腐剂检测报告;"天然 95%"必须有成分比例声明;"无动物试验"必须有 SCCS 认证。**无依据宣称 = 违规,2024-2025 意大利 / 法国专项执法重点**。

## 何时做 / 何时不做

**符合以下任一条件,立刻做**:
- ✅ 你在 Amazon EU 销售化妆品(任何类目)
- ✅ 你 2026 年后还要继续卖
- ✅ 你的产品含染发剂、烫发剂、美白、防晒、纳米材料等高风险成分
- ✅ 你重视 listing 长期稳定性

**暂缓**:
- ⚠️ 你 2025-2026 年内准备退出欧洲市场
- ⚠️ 你的产品是药品 / 食品 / 医疗器械(各有专项法规)
- ⚠️ 你的产品是二手 / 古董化妆品(罕见)

## 如果你只做一件事

**第一步先指定欧盟责任人 RP** —— 没有 RP,CPNP 通报都做不了,PIF 也没人维护,产品就是"无主"状态。RP 是**所有义务的起点**。可以选择**专业欧代服务商**(年费 300-1500 €),也可以指定你在欧盟的合作伙伴 / 律师事务所。**没有 RP,后续所有动作都做不了**。

**30 天内**:
- [ ] 产品清单 + 风险等级
