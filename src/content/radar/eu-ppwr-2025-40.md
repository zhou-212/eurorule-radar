## 一句话结论

如果你在欧盟销售带包装的商品,**2026-08-12 起**新规 PPWR (EU) 2025/40 全面适用 —— 包装的物质含量(重金属 + PFAS)、最小化(空置容积 ≤ 50%)、标签(运营主体 + 批次追溯)、可回收等级(A/B/C,2030 起)等多个维度全部进入执法。平台已陆续推送合规通知,**未完成全维度合规的在售商品可能被下架停售**。

## 适用范围

- **法规级别**:欧盟 **Regulation**(直接适用全境),**替代** 原 Directive 94/62/EC(自 2026-08-12 起,除部分附件条款另行规定)
- **覆盖包装类型**:所有投放欧盟市场的包装 —— 主要(销售)包装、次要(成组)包装、运输(三级)包装、服务包装(如外带容器)、复合包装
- **覆盖场景**:B2C + B2B;在线 + 线下;无论包装材料(塑料 / 纸 / 金属 / 玻璃 / 木材 / 复合材料)
- **不适用**(典型):医药、医用器械、敏感运输货物等另有专项法规的包装

## 关键时间线

- **2024-12-16**:欧盟理事会通过 PPWR
- **2024-12-19**:签署
- **2025-01-22**:在 Official Journal 发布
- **2025-02-11**:**正式生效** + 18 个月过渡期开始
- **2026-08-12**:**全面执行**(默认),各成员国可设最长 +5 年缓冲
- **2028-08-12**:**欧盟统一材质象形标识** 强制粘贴,逐步替代各成员国现行的垃圾分类符号
- **2030 / 2038 / 2040**:分阶段的可回收等级、再生原料配比、重复使用比例硬性目标

## 物质限制(2026-08-12 起)

| 限制 | 阈值 | 适用范围 |
|---|---|---|
| 重金属(铅、镉、汞、六价铬)总和 | ≤ 100 mg/kg | 所有包装 |
| 食品接触包装 — PFAS 单体 | < 25 ppb(靶向分析) | 食品接触材料(FCM) |
| 食品接触包装 — PFAS 总和 | < 250 ppb(靶向分析) | 食品接触材料(FCM) |
| 微塑料 | 受限(具体限值后续二级立法) | 全部 |

## 包装最小化(2026-08-12 起)

- **空置容积**:包装内部闲置空间 ≤ 包装总容积的 **50%**(气泡膜 / 泡沫填充 / 缓冲纸屑等全部计入)
- **禁止"过度包装"**:刻意放大产品视觉体积的双壁夹层、虚假底部、多层加厚非必要设计
- **大箱 + 小产品** 模式不再符合标准,需要按产品定制裁切

## 标签与溯源(2026-08-12 起)

- 包装上**必须**清晰标注:**运营主体完整名称 / 注册商号 / 实体联系地址**
- 配 **批次编码 / 序列号 / 二维码** 等可追溯标识
- 二维码需关联完整合规档案信息(为未来"数字产品护照 DPP"做准备)
- 平台与监管机构**同步收紧环保宣传标语**审核(避免"绿色 / 可降解 / 环保"等无依据宣称)

## 可回收 / 重复使用 / 再生原料目标(分阶段)

| 指标 | 2030 | 2038 | 2040 |
|---|---|---|---|
| 包装可回收等级要求 | A / B / C 三级并存 | 淘汰 C 级,仅 A / B 流通 | — |
| 电商运输包装重复使用占比 | 40% | — | 70% |
| 饮品零售可重复灌装包装 | 10% | — | 40% |
| 塑料包装再生原料最低占比 | 分品类设置 | — | 进一步上调(部分 ≥ 65%) |
| 包装废弃物总量 vs 2018 基线 | -5% | — | -15% |

## 你现在要做什么(6 个月窗口期)

1. **盘点全部出口欧盟产品的包装数据**。材质(塑料 / 纸 / 玻璃 / 金属 / 复合)、品类、重量、上游供应商、当前各国 EPR 注册状态 —— 全部建档
2. **立即启动 PPWR 框架下的 EPR 注册**。在任一欧盟成员国投放带包装商品,就要注册该国 EPR 包装类目(注:部分国家如荷兰因法规调整暂停,但仍可先排队);亚马逊九大欧盟核心站点全要,奥地利 / 丹麦 / 卢森堡如已开通跨境分销也需补
3. **送检有害物质**。重金属 + PFAS 都要,食品接触类必检;索取第三方实验室报告存档(合规核验核心凭证)
4. **重新设计外包装尺寸**。按"产品尺寸 + 50% 空置上限"反推包装,定制裁切,减少缓冲物料
5. **更新标签 + 准备溯源系统**。运营主体 + 批次 / 序列号 / 二维码;在 2026-08-12 前完成全部在售 SKU 切换
6. **关注 2028 统一象形图**。从 2028-08-12 起强制粘贴 EU 统一材质标识,意味着 2026-08-12 上线后 2 年内**还要再改一次标签**

- **高概率**:2026-08-12 后,Amazon EU / TEMU / 速卖通等平台对未完成 PPWR 合规的在售商品**统一下架停售**(平台已提前通知)
- **可能**:欧盟海关扣留不合规批次,新货无法入仓
- **可能**:成员国市场监管机构处罚 + 召回风险
- **明确:本卡片不构成法律意见**;具体品类(尤其食品 / 医疗 / 电气)可能适用额外规则,具体材质限值需查二级立法

## 常见问题

**Q1:PPWR 和原来的 94/62/EC 包装指令是什么关系?**
A:PPWR 是"升级版"。**法规**(Regulation)直接在全境适用,不需要各成员国转法;**指令**(Directive)需要各国转法。PPWR 自 2026-08-12 起替代 94/62/EC(部分条款除外)。

**Q2:我已经做了德国 LUCID 包装 EPR 注册,还要再做什么?**
A:LUCID 注册是 **EPR 框架**下的具体国家注册号,PPWR **也要求 EPR 注册**,所以 LUCID 这步**不浪费**。但 PPWR 还**叠加**了物质限值、最小化、标签、回收等级等要求,这些是 EPR 之外的新工作。

**Q3:卖家在欧洲没有公司,怎么注册?**
A:可以委托**本土授权代表**(Authorised Representative)代为承担生产者义务。多数国家法规明确允许,服务市场上有很多合规服务商(万理晴 / SGS / TÜV 等)提供打包服务。

**Q4:我卖二手 / 古董,适用吗?**
A:和 GPSR 类似,PPWR 对二手商品和"需要维修的二手物品"有部分豁免,但具体看你的产品形态和"投放市场"方式。保险起见咨询合规顾问。

**Q5:亚马逊在 2026-08-12 之前就强制下架不合规商品吗?**
A:平台已提前通知,具体执行节奏由平台决定。根据 2025-2026 多家媒体报道(亚马逊 / TEMU / 速卖通),**临近 8 月 12 日前 2-3 个月会进入集中审核期**,**建议 2026-06 之前完成全部合规工作**。

**Q6:PFAS 限制 25 ppb 怎么测?**
A:需要通过**靶向 PFAS 分析**(targeted PFAS analysis),一般送 SGS / TÜV / Intertek 等第三方实验室。聚合物 PFAS(polymer PFAS)不在 25/250 ppb 限制内,但需要单独判定。
