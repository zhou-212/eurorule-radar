## 一句话结论

**2026-09-02 起,所有在欧盟市场销售的食品接触材料**(厨具、餐具、食品包装、婴儿奶瓶、咖啡杯、烧烤工具等)**必须符合 FCMA(EU) 2024/1808** —— 强制 DoC + 物质限制 + 安全评估 + 标签 + 回收说明。亚马逊、欧盟海关、成员国市场监管机构都将执行。**罚款 10000-100000 €**,listing 下架。

## 这是啥

EU 2024/1808 食品接触材料法规(FCMA, Food Contact Materials and Articles Regulation)由欧盟理事会与议会通过,**2024-10-10 签署**,**2025-02 生效**,**2026-09-02 起强制实施**。新规**替代**:

- **EC 1935/2004**(食品接触材料框架法规)
- **EU 10/2011**(塑料专项)
- **EC 282/2008**(回收塑料)
- **EU 2018/213**(涂层 / 印刷油墨 BPA)

**FCMA 整合所有食品接触材料的统一规则**,从"分散立法"变成"一部大法",适用范围比之前**广得多**。

## 适用 vs 不适用

**适用**(典型类目,2026-09-02 起强制 FCMA):
- ✅ 厨具:锅、铲、勺、刀具、烤盘、烘焙模具
- ✅ 餐具:碗、盘、杯、叉、勺、筷
- ✅ 食品包装:外卖盒、保鲜盒、烘焙纸、烤箱纸
- ✅ 饮料容器:水杯、咖啡杯、保温杯、酒杯
- ✅ 婴儿用品:奶瓶、奶嘴、辅食碗、儿童餐盒
- ✅ 储存容器:玻璃罐、塑料罐、金属罐
- ✅ 印刷油墨 / 涂层:包装外印的油墨、锅内涂层
- ✅ 烧烤工具:烤架、烤签、刷子(食品接触部分)

**不适用**(FCMA 范围外):
- ❌ 食品本身
- ❌ 化妆品 / 药品 / 饲料(各有专项法规)
- ❌ 固定式公共设施(如公共饮水机内部组件)
- ❌ 复古 / 二手食品接触材料(需特殊处理)

## 为什么跟你有关

**亚马逊公开公告**(Seller Central 2025-11):
> "All food contact materials sold in the EU must comply with Regulation (EU) 2024/1808 by September 2, 2026. Sellers must provide Declaration of Conformity and material safety documentation through Seller Central."

**没做的后果**:
- 亚马逊后台 `Food contact material compliance` 字段空 → listing 限流
- 物质检测报告缺失 → 亚马逊发"补充资料"邮件
- 持续 60 天不合规 → listing 在欧盟市场不可见
- 欧盟海关抽查 → 罚款 10000-100000 €
- 成员国市场监管机构(德国 BVL / 法国 DGCCRF / 意大利 MOH)强制召回

**15 个月窗口期特别紧的原因**:
- 食品接触材料**品类多**(塑料、纸、金属、玻璃、陶瓷、油墨、涂层、橡胶)
- 物质限制**复杂**(重金属、PFAS、邻苯二甲酸盐、双酚、芳香胺)
- 检测项目**比普通电子产品多 1-2 倍**
- Notified Body 业务量激增,排期 3-6 周

## FCMA vs 旧 1935/2004 关键差异

| 维度 | 旧 1935/2004 框架 | 新 FCMA 法规 | 影响 |
|---|---|---|---|
| 适用材料 | 各成员国转化立法 | 直接适用全境 27 国 + 监管 | 统一标准 |
| 物质 | 仅框架 + 塑料专项 | 整合 8 大材料 + 18 类物质 | 检测项增加 |
| DoC | 自愿 | 强制(品牌名 + 适用法规 + 测试报告) | 新增要求 |
| 标签 | 各成员国自定 | 欧盟统一标签(玻璃 / 陶瓷回收标志 + 食品接触标志) | 包装改版 |
| 安全评估 | 仅塑料必需 | **所有材料必需** | 增加评估成本 |
| 数字化 | 无 | 数字产品护照 DPP(2030 起分阶段) | 未来要再改 1 次 |
| 处罚 | 各成员国自定 | **统一罚款框架**(10000-100000 €) | 风险明确 |

## 5 步行动框架(总预算 3000-8000 €)

### 第 1 步:类目确认 + 材质分类(预算 0 €,1 周)

**做什么**:
- 列出**所有投放欧盟市场的食品接触材料**,按 8 大类分类
- 8 大类:塑料、纸 / 纸板、金属、玻璃、陶瓷、橡胶、印刷油墨、涂层
- 每类记录:SKU / 类目 / 材质 / 重量(g) / 体积(cm³) / 生产商 / 投放国 / 是否接触食品 / 接触温度范围

**工具**:
- Excel / Google Sheets
- 字段:SKU / 类目 / 材质 / 重量 / 体积 / 生产商 / 投放国 / 食品接触 / 温度范围

**避坑**:
- 别只列"主产品"——食品接触**所有**接触部分(杯盖、密封圈、涂层、油墨)都要算
- 跨 SKU 相似材料 → 单独建一行,标注"共用"
- 油墨 / 涂层单独检测(很多卖家漏掉)

### 第 2 步:物质检测(预算 2000-6000 €,3-6 周)

**必检项**(基于 FCMA 第 5-9 条 + 附件):
- **重金属**:铅、镉、汞、六价铬、镍 ≤ 迁移限量(SML)
- **PFAS 全氟烷基物质**:≤ 25 ppb(食品接触 / 靶向分析)
- **邻苯二甲酸盐**:DEHP / DBP / BBP / DIBP ≤ 0.1%(重量比)
- **双酚 A (BPA)**:涂层 / 塑料 ≤ 0.05 mg/kg(婴儿用品禁)
- **芳香胺**:染色 / 印花材料 ≤ 0.1 mg/kg
- **REACH 高度关注物质(SVHC)**:≤ 0.1%(按欧盟 ECHA 清单)
- **特定迁移总量(OML)**:60 mg/kg(模拟液浸泡测试)

**找谁做**:
- Notified Body:SGS / TÜV SÜD / Intertek / Bureau Veritas / Eurofins
- 选有 FCMA 实操经验的(2026 年 1 月起业务量激增,排期长)
- 报价区间:2000-6000 €(首批 5-10 个 SKU 约 4000 €)

**关键**:
- 检测报告**保留 5 年**,亚马逊和欧盟市场监管机构抽查时需要
- 报告要**含品牌名 + SKU + 检测日期 + 检测机构印章**
- 跨 SKU 相似材料可以**合并检测**(节省 30-50% 成本)
- 油墨 / 涂层 / 密封圈等**每种材质**都要单独测

### 第 3 步:DoC 起草 + 签发(预算 0-500 €,1 周)

**DoC 必备内容**(FCMA 第 15 条 + 附件 III):
- 制造商 / 欧盟授权代表信息
- 产品名 / 型号 / SKU
- 适用法规清单(EU 2024/1808 + 协调标准)
- 检测报告引用(机构 + 报告号 + 日期)
- 安全评估摘要(适用材料 / 用途 / 接触条件)
- 签发日期 + 签发人签名
- 物质限制符合性声明(8 大类分别说明)

**自起草 vs 委托**:
- 简单 SKU(单一材料 / 单一用途):自起草(用模板)
- 复杂 SKU(多种材料 / 多种用途):委托合规顾问(300-500 € / SKU)
- 模板可参考英国标准协会(BSI)或德国 TÜV 提供的 FCMA DoC 模板

### 第 4 步:标签印制(预算 500-2000 €,1-2 周)

**必印内容**(FCMA 第 12-14 条 + 各国延伸):
- **食品接触标志**:欧盟统一"刀叉杯"象形(高度 ≥ 5mm)
- **材质回收标志**:三角循环符号 + 字母缩写(PET / PP / PE / 玻璃 / 纸)
- **批次 / 生产日期**:便于追溯,通常印为"LOT:XXX"或日期
- **运营主体**:谁投放市场(品牌名 + 欧盟地址)
- **使用条件**:微波 / 洗碗机 / 烤箱 / 冷冻适用温度(若适用)
- **警告语**:高温烫手 / 婴儿使用 / 一次性(若适用)

**印刷方式**:
- **小批量(< 1000 件)**:贴纸 + 手工贴(0.5-1 € / 件)
- **中批量(1000-10000 件)**:产品本体直接印刷(0.1-0.5 € / 件)
- **大批量(> 10000 件)**:模具内注塑 / 烫印(0.05-0.2 € / 件)

**避坑**:
- 别只用英语 / 中文——**至少 5 国语言**标签(EN / DE / FR / IT / ES)
- 印刷前**打样确认**(包装厂提供)
- 油墨 / 涂层材料单独标注(很多卖家漏掉)

### 第 5 步:亚马逊后台 + 详情页同步(预算 0 €,1 周)

**后台字段**(Seller Central):
- `Manage Inventory → Compliance → Food contact material compliance`
- 每个 SKU 填:类目 / 材质 / 是否可接触食品 / 接触条件 / 检测报告号
- 上传:DoC 扫描件 + 检测报告(若适用)

**详情页更新**:
- `Compliance` 板块加:FCMA 合规说明 + 多语种版本
- 主图区域可加"Food safe"或"EU food grade"标识(亚马逊官方允许)
- 关键文档:DoC PDF(可在详情页 Compliance 板块提供下载)

## 4 个常见误区(基于公开报告)

**误区 1:"我的产品有 LFGB 认证(德国) = 欧盟合规"**
LFGB 是德国本地食品接触材料法规,FCMA 是**欧盟统一法规**。LFGB 认证**包括大部分 FCMA 要求**,但**不能等同**:FCMA 增加了油墨 / 涂层 / 双酚 A 限制,LFGB 不全部覆盖。**有 LFGB 还要做 FCMA 补充**。

**误区 2:"FDA 认证(美国)能在欧盟用吗"**
不能。FDA 是美国 FDA 食品接触材料规则,FCMA 是欧盟规则。**两个独立的体系**。**双地区销售 = 两套检测报告**。

**误区 3:"15 个月够了,不急"**
错。FCMA 物质限制复杂,**单 SKU 检测 3-6 周**。多材质 / 多 SKU 累计 6-10 周。**等到 6 月才下单,可能 9 月还没拿到报告**。建议 2026-Q1 启动。

**误区 4:"亚马逊没催就不急"**
亚马逊的合规审核是**分批次的**——你当前没被轮 ≠ 安全。等到被轮,改已经来不及。**2026-09-02 之前 2-3 个月会进入集中审核期**。

## FAQ

### FCMA 跟 LFGB / FDA 什么关系?

- **FCMA**:欧盟统一法规(2026-09-02 实施)
- **LFGB**:德国本地食品接触材料规则(已生效)
- **FDA**:美国 FDA 食品接触材料规则(已生效)

LFGB 认证**包括大部分 FCMA 要求**,但**FCMA 增加了油墨 / 涂层 / 双酚 A 限制**。**有 LFGB 还要做 FCMA 补充**。FDA / 欧盟不互通。

### 物质检测找哪家最稳?

SGS / TÜV SÜD / Intertek / Bureau Veritas / Eurofins 都有 FCMA 实操经验。建议**选有欧盟本地实验室**的(德国 TÜV SÜD / 法国 BV / 意大利 Eurofins),报告欧盟市场认可度更高。

### 15 个月够吗?时间紧怎么压缩?

预算够的话,选**合并检测**(多种材质) → 报告 4-6 周出;**预打样标签**(第 4 步同步做) → 节省 1 周;**后台字段预先填**(第 5 步提前) → 9 月前再上检测报告链接。

### 婴儿用品(奶瓶 / 奶嘴)有额外要求吗?

有。FCMA 对**婴儿和儿童食品接触材料**有**额外限制**:
- 双酚 A(BPA)**禁止**
- 邻苯二甲酸盐**严格限制**(≤ 0.01%)
- PFAS 限值更严
- 必须标注"适合儿童使用"或年龄范围

### 检测报告能被亚马逊 / 海关抽查时认吗?

Notified Body 出具的报告 + 欧盟本地实验室 + 包含品牌名 / SKU / 检测日期 = 100% 被认。**小作坊或非 Notified Body 的报告,抽查时被拒**。

### 包装上只印英语标签可以吗?

不行。FCMA 要求**投放国本地语言**。如果投放到 DE / FR / IT / ES / NL,标签至少要 EN + DE + FR + IT + ES 5 国语言。**多语种标签成本只增加 5-10%**,别省。
