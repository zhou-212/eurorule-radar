## 一句话结论

法国 EPR 涉及**多个回收体系**:包装(Citeo、Syderep 等)、电器(生态系统如 Ecologic)、电池、纺织品、家具、轮胎、纸张等。亚马逊法国站要求所有相关类目卖家**必须登记并提供 EPR 编号**,否则将被限制 FBA 入仓 / 销售。

## 适用范围

- **站点**:Amazon FR(法国站)
- **卖家类型**:任何把受 EPR 监管商品销往法国的卖家
- **类目覆盖**:包装、电气电子、电池、纺织品、家具、轮胎、纸张、化学品、印刷品(各对应不同组织)
- **不适用**:完全虚拟商品

## 关键时间线

- **2022-01-01**:法国 Anti-Waste Law(AGEC)主要条款生效
- **2022 起**:Amazon FR 在后台强制要求 EPR 编号(类目不同,要求时间不同)
- **2024-2025**:平台对未填写 EPR 编号的卖家,持续进行**库存停运 / Listing 限制**操作
- **持续**:ADEME / Citeo 等机构对违规进行罚款风险

## 你现在要做什么

1. **判断你的商品属于哪几类 EPR**(包装、电气、电池、纺织、家具、轮胎、纸张);同一商品可能触发多项。
2. **到对应机构注册**:包装 → Citeo;电气 → Ecologic / Screlec 等;电池 → Screlec / Corepile;纺织 → Refashion。每类需要单独申请。
3. **把 EPR 编号填入 Amazon FR 后台**。各 EPR 类目有独立字段,例如 "EPR Packaging"、"EPR WEEE"、"EPR Battery"。
4. **包装与说明书添加 Triman 标识 + 唯一识别符**:Triman 标识按 ADEME 指南排版;唯一识别符由注册机构提供。
5. **法语翻译**:包装或说明必须**有法语版本**,Triman 与回收说明要在法语文案内。
6. **跨平台同步**:Cdiscount、Fnac 等其他法国平台也强制 EPR,不要只在 Amazon 上做。

## 不处理的潜在影响

- **可能**:Amazon FR Listing 限流 / FBA 库存停运
- **可能**:ADEME / Citeo 进行合规审计,最高罚款可达**数万欧元**
- **可能**:消费者诉讼风险("信息告知不充分")
- **明确:本卡片不构成法律意见**

## 常见问题

**Q1:Triman 标识必须印在包装上吗?**
A:是的。AGEC 实施后,Triman 标识 + 唯一识别符需印刷在产品或包装上(某些类目可电子化告知,但有条件)。

**Q2:我产品类别多,需要每个都申请吗?**
A:是的。每个触发 EPR 的类目必须单独注册,亚马逊每个类目字段也独立。

**Q3:EPR 编号和法国 EPR 税(SIREN)是一回事吗?**
A:不是。EPR 编号由回收组织发放,证明你已加入回收体系;SIREN 是公司识别号,两者独立但可能都在合规页面用上。

**Q4:只有 FBA 卖家需要 EPR 吗?**
A:FBM 也需要,只要在 Amazon FR 卖就触发要求,亚马逊对所有卖家统一强制。
