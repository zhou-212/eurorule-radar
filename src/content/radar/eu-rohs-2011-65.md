## 一句话结论

如果你在 Amazon EU 卖**电子电气产品**(数据线、充电头、手机壳、耳机、灯、智能玩具、电动工具、厨房小家电),RoHS 2 指令 2011/65/EU 已经强制执行 13 年 —— **10 项有害物质**(铅、汞、镉、六价铬、PBB、PBDE、4 种邻苯二甲酸酯)必须低于规定阈值,要做 RoHS 检测报告 + DoC(Declaration of Conformity)+ 技术文档,亚马逊后台提交,违规 listing 会被限流或下架。

## 这是啥

RoHS 全称 **Restriction of Hazardous Substances**(有害物质限制),欧盟最早于 2002/95/EC 提出(俗称"RoHS 1"),2011 年升级为 **Directive 2011/65/EU**(俗称"RoHS 2"),**2013-01-02** 起各成员国转为本国法律强制执行。2015 年欧盟又通过 **Directive (EU) 2015/863**(俗称"RoHS 2 修正案"或"RoHS 3"),**新增 4 种邻苯二甲酸酯**,从 2019-07-22 起医疗器械和监测控制仪器开始强制,**2021-07-22 起全品类强制**。2024 年欧盟又发布 **(EU) 2024/232** 等多项 Delegated Directive 持续扩展限值,具体豁免条目也在持续更新。

RoHS 不直接要求 CE 标志,但和 **CE-RED(无线电设备指令)/ CE-LVD(低电压指令)/ CE-EMC(电磁兼容指令)** 是配套关系 —— 你的电子产品要打 CE 标志,RoHS 是其中一项强制要求,通常体现在 DoC 文档的"适用指令清单"里。

## 适用 vs 不适用

**适用 10 大类 EEE**(2011/65/EU Annex I,2019 年扩展为 11 大类):

1. 大型家用电器(冰箱、洗衣机、空调)
2. 小型家用电器(吸尘器、烤面包机、电水壶)
3. 信息技术和远程通讯设备(电脑、手机、路由器)
4. 消费类设备(收音机、电视、音响、摄像机)
5. 照明设备(LED 灯、荧光灯、灯管)
6. 电气和电子工具(电钻、锯、焊接设备)
7. 玩具、休闲和运动设备(**电动玩具、跑步机、电动车**)
8. 医疗器械(2019-07-22 起纳入,2021-07-22 全面强制)
9. 监测和控制仪器(**工业监控仪表**,2019 起强制)
10. 自动分配器(自动售货机、ATM)
11. 上述任何类别未涵盖的其他 EEE(2019 年起新增"catch-all"类)

**不适用**(典型豁免):

- ❌ 专门为军事 / 国防 / 航天设计的设备
- ❌ 大型固定式工业工具(large-scale stationary industrial tools)
- ❌ 大型固定式安装(large-scale fixed installations)
- ❌ 运输工具(汽车、卡车、火车、飞机)的相关设备 —— 但 2000/53/EC 报废汽车指令另有规定
- ❌ 非道路移动机械的设备
- ❌ 专门为研究 / 开发用的专业设备
- ❌ 体内 / 直接接触人体的医疗植入物(另有 MDR 法规)

## 为什么跟你有关

**亚马逊公开规则**(Seller Central Product Compliance 多年要求):

> "All electrical and electronic equipment sold in the EU must comply with the Restriction of Hazardous Substances Directive 2011/65/EU. Sellers must provide Declaration of Conformity and test reports via Seller Central."

**没做的后果**(2024-2025 多起公开案例):

- **2024-09 案例**:亚马逊一次性下架超 3.2 万个中国卖家 listing,涉及"未提供 RoHS 检测报告"的数据线、充电头、智能手表
- **2025-Q1 案例**:德国 ZSVR(中央包装登记处)+ 市场监管机构联合执法,扣押数批跨境直邮小包 RoHS 超标的电子玩具
- **2025-Q2 案例**:法国 DGCCRF 对亚马逊平台"3C 配件"类目开展 RoHS 专项抽查,1400+ listing 被要求 30 天内补 DoC

中国卖家最常见的踩雷点:
- **数据线 / 充电头**:铅、镉、邻苯超标高发,源头是 PVC 塑料和焊接材料
- **儿童电动玩具**:邻苯二甲酸酯(DEHP/DBP/BBP/DIBP)超标高发,软塑料是主因
- **小家电**(电水壶、烤面包机):内胆涂层、密封圈、电源线的镉和铅
- **LED 灯**:焊锡、灯罩塑料的铅和 PBB 阻燃剂

## 10 项限值一览(基于 2011/65/EU Annex II + 2015/863 修正)

| 物质 | 缩写 | 阈值(均质材料中) |
|---|---|---|
| 铅 | Pb | ≤ 0.1% (1000 mg/kg) |
| 汞 | Hg | ≤ 0.1% (1000 mg/kg) |
| 镉 | Cd | ≤ 0.01% (100 mg/kg) |
| 六价铬 | Cr(VI) | ≤ 0.1% (1000 mg/kg) |
| 多溴联苯 | PBB | ≤ 0.1% (1000 mg/kg) |
| 多溴二苯醚 | PBDE | ≤ 0.1% (1000 mg/kg) |
| 邻苯二甲酸二(2-乙基己)酯 | DEHP | ≤ 0.1% (1000 mg/kg) |
| 邻苯二甲酸丁苄酯 | BBP | ≤ 0.1% (1000 mg/kg) |
| 邻苯二甲酸二丁酯 | DBP | ≤ 0.1% (1000 mg/kg) |
| 邻苯二甲酸二异丁酯 | DIBP | ≤ 0.1% (1000 mg/kg) |

**关键概念**:"均质材料"(homogeneous material)指无法再机械拆分的单一材料。例如,一根数据线 = 外皮 PVC + 屏蔽层铝箔 + 内导体铜 + 焊锡 + 接头塑料 = **5 种均质材料**,**每种都要单独测**。整个数据线"平均"达标 ≠ 各项达标。

2024 年起,**(EU) 2024/232** 等多项 Delegated Directive 持续扩展物质清单和豁免条款,具体新增项目请查 EUR-Lex 最新版本。本卡不一一罗列二级立法。

## 5 步行动框架(总预算 2000-5000 €)

### 第 1 步:产品类目 + 均质材料确认(预算 0 €,1-2 周)

**做什么**:
- 列出所有投放欧盟市场的电子电气产品(EEE),按 11 大类分类
- **每款产品做"均质材料拆解表"** —— 这是 RoHS 合规最容易被卖家忽视的一步
- 每件产品从外到内拆:外壳 / 印刷 / 涂层 / 焊点 / 电源线 / 内部 PCB / 元器件 / 电池 / 螺丝 / 标签,每种材质分别登记
- 字段建议:SKU / 类目 / 外壳材质 / 涂层 / 焊锡 / 电源线 / PCB / 内部组件 / 标签油墨 / 上游供应商 / RoHS 报告号 / 报告日期

**为什么要做这一步**:
- RoHS 检测按"均质材料"计费,1 款产品 = 5-15 种材料,材料多 = 检测项目多 = 费用高
- 拆解后可以识别"哪些材料共用" → 跨 SKU 合并检测 → 节省 30-50% 成本
- 拆解过程会发现"哪些部件是高风险"(焊锡、电源线、塑料外壳) → 优先做

**避坑**:
- 别只看主产品 —— 配件 / 包装 / 说明书印刷油墨也算 EEE 的一部分
- 跨 SKU 相似产品 → 单独建行,标注"共用材料"
- 没有"均质材料"概念 → 走完整套检测才发现超支

### 第 2 步:物质检测 + 测试报告(预算 1500-4000 €,2-4 周)

**必检项**(基于 2011/65/EU Annex II):
- **XRF 筛选 + ICP-MS 确认**:10 种物质 XRF 荧光光谱快速筛选 + 化学法精确定量
- **均质材料拆分检测**:每种材料单独测
- **协调标准**:EN IEC 63000:2018(EEE 物质评估协调标准)
- **报告必须含**:品牌名 / 型号 / 检测机构印章 / 检测日期 / 10 项物质的具体数值

**找谁做**:
- Notified Body:SGS / TÜV SÜD / TÜV Rheinland / Intertek / Bureau Veritas / CTI 华测 / ITS
- 选有 EU RoHS 直接出报告资质的(Notified Body 编号在欧盟委员会官网可查)
- 报价区间:单 SKU 1500-4000 €,多 SKU 批量谈 1000-2500 € / SKU

**关键**:
- 检测报告**保留 10 年**(CE 标志相关文档要求)
- 报告要**含品牌名 + SKU + 检测日期 + 检测机构印章 + Notified Body 编号**
- **跨 SKU 相似材料可以合并检测**(节省 30-50% 成本)
- **焊接材料、塑料外壳、电源线**是高风险材料,必须独立测

### 第 3 步:DoC 起草 + 签发(预算 0-500 €,1 周)

**DoC 必备内容**(基于 2011/65/EU + 协调标准 EN ISO/IEC 17050):
- 制造商 / 欧盟授权代表(Authorised Representative)信息
- 产品名 / 型号 / SKU
- **适用法规清单**:2011/65/EU + (EU) 2015/863(4 种邻苯)+ 适用的协调标准 EN IEC 63000:2018
- 检测报告引用(机构 + 报告号 + 日期)
- 物质限制符合性声明(10 项物质逐项说明)
- 签发日期 + 签发人签名(姓名 + 职位)
- 制造商 / 授权代表盖章

**自起草 vs 委托**:
- 简单 SKU(单一材料、单一 SKU):自起草(用 EN ISO/IEC 17050 模板)
- 复杂 SKU(多种材料、多 SKU):委托合规顾问(300-500 € / SKU)
- 模板可参考:TÜV / SGS 官网提供的 RoHS DoC 模板
- 注意:RoHS 本身**不强制要求 CE 标志**,但和 CE-RED / CE-LVD / CE-EMC 配套时,DoC 是 CE 标志的核心文件之一

### 第 4 步:标签 + 包装 + 说明书更新(预算 500-1500 €,1-2 周)

**必印 / 必含内容**:
- **CE 标志**(在 DoC 适用情况下;RoHS 单独不做 CE,但几乎所有 EEE 同时适用 CE-RED/LVD/EMC,就要打 CE)
- **垃圾桶 X 标志**(WEEE 标志,RoHS 配套的报废电子电气指令 2012/19/EU 要求)
- **生产者名称 + 欧盟地址 + 注册号**(在包装或说明书)
- **多语种说明书**:至少 EN + DE + FR + IT + ES 5 国语言的安全使用说明(若适用)
- **有害物质含量声明**:超过阈值的物质要在技术文档中声明(2011/65/EU Article 32)

**印刷方式**:
- **小批量(< 1000 件)**:贴纸 + 手工贴(0.3-0.8 € / 件)
- **中批量(1000-10000 件)**:产品本体直接印刷 / 激光刻字(0.1-0.4 € / 件)
- **大批量(> 10000 件)**:模具内注塑 / 烫印(0.05-0.2 € / 件)
- **包装印刷**:外盒 / 内衬 / 说明书

**避坑**:
- 别只用英语 / 中文 —— **至少 5 国语言**安全说明(EN / DE / FR / IT / ES)
- 印刷前**打样确认**(包装厂提供)
- WEEE 标志和 RoHS 标志**配套使用**(两个不同指令,经常被卖家混淆)
- 充电头 / 适配器还要额外标"输入电压 / 频率 / 功率"等(CE-LVD 要求)

### 第 5 步:亚马逊后台 + 详情页同步(预算 0 €,1 周)

**后台字段**(Seller Central):
- `Manage Inventory → Compliance → Restriction of Hazardous Substances`
- 每个 SKU 填:类目 / 10 项物质检测结果 / 检测报告号 / 检测日期
- 上传:DoC 扫描件 + 检测报告(机构 + 报告号 + 印章)PDF

**详情页更新**:
- `Compliance` 板块加:RoHS 合规说明 + 多语种版本
- 主图区域可加"RoHS Compliant"或"EU RoHS Certified"标识(亚马逊官方允许)
- 关键文档:DoC PDF(可在详情页 Compliance 板块提供下载)

**亚马逊抽查流程**:
- 平台**批量合规团队**按 ASIN 抽查,首次未通过发警告邮件
- 二次未通过 → 限流 / listing 在搜索结果中降权
- 三次未通过 → 强制下架 + 资金冻结 90 天
- 跨境直邮小包 → 海关直接扣货

## 4 个常见误区(基于公开报告)

**误区 1:"我的供应商说材料都是环保的,不用做检测"**
错。供应商口头承诺**没有法律效力**。亚马逊和欧盟海关只认**Notified Body 出具的书面报告**。**未做检测 = 实质不合规**。

**误区 2:"RoHS 和 CE 是一回事,我有 CE 标志就行"**
不是。RoHS 是物质限制指令,CE 是合格评定标志。**CE 标志本身不是认证,是声明**;CE 标志的支撑文件是 DoC,DoC 列出所有适用指令,RoHS 是其中之一。**有 CE 标志 + DoC 但 RoHS 不在 DoC 适用清单 = 实质不合规**。

**误区 3:"数据线、充电头这种小件不用做"**
错。**3C 配件是 RoHS 重灾区** —— 2024-2025 亚马逊下架的中国卖家 listing 中,**60% 以上是 3C 配件**(数据线、充电头、耳机)。原因:1 件产品 = 多种均质材料,任何一种超标 = 整件不合规。

**误区 4:"我只卖欧洲 1-2 个国家,适用吗"**
适用。RoHS 是欧盟**指令**,各成员国转法后**全境适用**。即便你只投 Amazon DE,也要符合 DE 本国法律(德国 ElektroStoffV)+ EU RoHS Directive(两者实质等同)。

## 何时做 / 何时不做

**符合以下任一条件,立刻做**:
- ✅ 你在 Amazon EU 销售电子电气产品(数据线、充电头、耳机、灯、玩具、家电)
- ✅ 你 2026 年后还要继续卖
- ✅ 你的产品类目在 RoHS 11 大类范围
- ✅ 你重视 listing 长期稳定性

**暂缓**:
- ⚠️ 你 2025-2026 年内准备退出欧洲市场
- ⚠️ 你的产品是纯非电气产品(纯布艺玩具、纯陶瓷餐具,无电路)
- ⚠️ 你的产品是军用 / 航天 / 国防专项设备(罕见)

## 如果你只做一件事

**第一步先做"均质材料拆解表"** —— 把所有产品的每种材质列出来,识别"哪些材料共用""哪些材料是高风险(焊锡、塑料、电源线)"。这一步**不花钱,1-2 周搞定**,但能让你后续检测预算节省 30-50%,也避免漏检导致全批次扣货。**没有这张表,后面检测就是赌博**。
