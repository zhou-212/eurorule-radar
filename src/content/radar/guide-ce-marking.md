## 一句话结论

**CE 标志 = 欧盟的"商品合格证"**,你(制造商 / 品牌方)自己声明"我的产品符合欧盟健康 / 安全 / 环保要求",**几乎所有非食品电子、玩具、机械、医疗器械、个人防护装备必须有**。打的位置:产品本体、包装、说明书 **3 选 1**(建议都打),**高度 ≥ 5mm,比例不变形**;**用错标志(尤其是 China Export)罚款可达 5 万欧元 + listing 永久下架**。

## 这是啥

CE 是法语 "Conformité Européenne" 的缩写,意思是"欧洲合格"。它是欧盟在 1993 年推出的产品合格标志 —— 相当于一个**"我承诺安全"的小印章**,由**制造商自己贴**(不是政府发的),但要承担法律责任。

类比一下:

- **CE** ≈ 欧盟版"商品质量合格证"
- 美国的类似标志是 **FCC**(电子产品)和 **UL**(电气安全)
- 英国的(脱欧后)是 **UKCA**(UK Conformity Assessed)
- 中国的(出口用)是 **CCC**(中国强制性认证)

**欧盟要求**:产品上**必须能直接看到 CE 标志**(消费者、监管机构、亚马逊审核员都能验证)。所以打在哪、怎么打、有多高,都有严格规定。

## 为什么跟你有关

**1. 不打 CE = 不能在欧盟合法销售**

亚马逊对缺 CE 的产品会**直接下架或拒收**;欧盟海关对没 CE 的进口商品**直接扣留**;市场监管机构查到,**最高罚款 5 万欧元 + 召回风险**。

**2. 打错 CE = 比不打还糟**

最常见的错:把 **"China Export"**(中国出口)标志**当成 CE 标志**。这俩标志**长得像双胞胎** —— 都是两个字母 + 类似结构,但**中间间距不同**:

- **CE**(欧盟合格证):两个字母之间**正常间距**,C 的开口朝右
- **China Export**(中国出口):两个字母之间**间距更宽**,C 的开口朝右但弧度略小

欧盟法院 2019 年有过一个判例:某中国卖家把 China Export 标志当 CE 打,被认定"故意误导",罚款 €15,000 + 召回所有在售产品。

**3. 打了 CE 但没 DOC = 等于没打**

DOC = Declaration of Conformity(符合性声明)。这是一份**你签的"自我声明文件"**,写清楚:

- 你的公司名 + 地址
- 产品名 + 型号
- 你做了哪些测试(列出标准号)
- 符合哪些欧盟指令(EMC / LVD / RED / Toy / MDD / MDR…)
- 声明日期 + 签字人

CE 标志 + DOC 是**配套的**。亚马逊 / 监管机构任何一个来查,你要**两个都能拿出来**。

## 哪些产品必须有 CE?(不完全清单)

以下类目**强制需要 CE**(不是穷尽,具体看你产品):

| 类目 | 典型产品 | 关联欧盟指令 |
|---|---|---|
| **电子电气** | 手机、充电器、笔记本、智能音箱、LED 灯 | EMC(电磁兼容)+ LVD(低电压)+ RoHS(有害物质) |
| **带无线** | 蓝牙耳机、智能手表、WiFi 路由器 | + RED(无线电设备指令) |
| **玩具** | 儿童玩具、毛绒、电动玩具 | Toy Safety Directive(玩具安全) |
| **机械** | 小型电动工具、缝纫机、3D 打印机 | MD(机械指令) |
| **医疗** | 口罩、血压计、按摩仪、护腰带 | MDR(医疗器械)+ IVDR(体外诊断) |
| **个人防护** | 头盔、护目镜、防护手套、KN95 口罩 | PPE Regulation(个人防护装备) |
| **燃气器具** | 燃气热水器、户外燃气炉 | GAD(燃气器具指令) |
| **压力设备** | 压力锅、气压设备 | PED(压力设备指令) |
| **建材** | 水泥、地板、保温材料 | CPR(建筑产品法规) |

**不强制 CE**(部分豁免):

- **食品**(归食品法规管)
- **药品**(归药品法规管)
- **化妆品**(归化妆品法规 1223/2009)
- **纯手工 / 二手 / 古董**(部分豁免)
- **军用 / 航空航天**(单独规则)

**简单判断**:你的产品**不是食品、不是药品、不是化妆品,还有外壳 / 电路 / 电池 / 机械部件** → **大概率要 CE**。

## 在哪打 CE?(3 选 1,建议都打)

欧盟法规规定 CE 标志可以打的位置:

1. **产品本体**(首选 — 最直接,消费者 / 监管机构一眼能看到)
2. **包装上**(本体空间不够时,比如小饰品、纽扣电池)
3. **说明书 / 随附文件**(本体和包装都不行时,比如易磨损产品)

**实务建议**:**3 个位置都打**。理由:

- 产品本体 CE → 海关 / 监管检查最直接
- 包装 CE → 亚马逊 FBA 入仓时方便扫描核验
- 说明书 CE → 消费者有疑问时可以翻阅
- **3 个位置都打,即使其中一个磨损 / 丢失,还有另外 2 个兜底**

## CE 标志的"硬性技术要求"

欧盟在 **Decision 768/2008/EC + Regulation (EC) 765/2008** 里规定 CE 标志的样式:

1. **字母 "C" 和 "E" 必须清晰、可辨认** — 不能变形、不能断笔
2. **高度(高度 = 字母 E 的高度)≥ 5mm**(小产品可以缩小,但**不能小于 5mm**)
3. **比例不变**:C 和 E 之间的间距是固定的(CE 内部间距 ≈ 1/5 E 高度),**不能拉宽也不能压扁**
4. **颜色对比度足够**:CE 标志可以用**任意颜色**(实际都是黑或白底黑字),但**必须和背景形成明显对比**
5. **不能遮挡**:CE 标志不能被其他图案 / 文字 / 装饰物盖住
6. **禁止缩写 / 修改**:不能打成"CE!" "C.E" "C~E" 等

```
标准 CE 标志(比例示意):

   ┌────────────────┐
   │  C  E           │   ← 字母高度 ≥ 5mm
   │  ↑  ↑           │   ← C-E 间距 = E 高度 × 1/5
   │  └──┘           │   ← 两个字母底边对齐
   └────────────────┘
```

**实操小技巧**:直接在 Open-Illustrations / Vecta.io / Iconfinder 等网站搜 "CE mark vector" 下载**官方标准矢量图**(SVG 格式,缩放不失真),**不要用 PNG 截图**(放大模糊,监管可能不认)。

## "China Export" 标志的致命陷阱

**这是新手最常踩的雷,务必认真看**。

中国出口行业有一个**民间自创的标志**叫 "China Export" — 长得和 CE 几乎一模一样,但**代表"中国出口"**(不是"欧盟合格")。两个标志对比如下:

| 特征 | ✅ 真正的 CE(欧盟合格) | ❌ China Export(中国出口,不是 CE) |
|---|---|---|
| 字母 | C 和 E | C 和 E(一样!) |
| **C-E 间距** | **正常(约 1/5 E 高)** | **明显加宽(约 4/5 E 高)** |
| C 的弧度 | 完整 | 弧度略小,像半个 C |
| 欧盟法律地位 | 官方合格标志(法规认可) | 民间标志(无法律地位) |
| 海关 / 亚马逊接受度 | 接受 | 视为"无标志" + 误导 |
| 罚款风险 | — | **€10,000-€50,000 + 召回** |

**常见误解**:

- "我用的就是 CE 标志,设计图就是 CE 啊" → **设计稿可能没问题,印刷厂用错图就完蛋**。**下单前要 PDF 矢量图打样确认**。
- "客户那边也是这么打的,大家都是这样" → **所有人都在犯同一个错,不代表就是对的**。海关一查一个准。
- "我打了 CE 标志 + DOC,应该没事" → **但你的 CE 标志其实是 China Export 标志**,等于**没打 + 虚假声明**,**责任更重**。

**自检方法**:

1. 量一下两个字母之间的间距(用尺子或 Illustrator)
2. 间距 ≈ E 高度 × 1/5 → 大概率是 CE
3. 间距 ≥ E 高度 × 1/2 → 大概率是 China Export(打错)
4. 不确定?**让欧代或认证机构帮你看**

## DOC(Declaration of Conformity)怎么写

DOC 是**你自己签的"自我声明"**,不需要政府盖章,但要承担法律责任。**亚马逊审核、海关清关、市场监管抽查**都可能要 DOC,所以**每个产品都要单独有一份**。

DOC 至少要包含(欧盟 768/2008/EC 规定):

1. **声明人信息**:公司名、地址、联系方式
2. **产品信息**:产品名、型号 / SKU、批次 / 序列号规则
3. **制造商信息**:同声明人或单独写(欧盟进口商 / 欧代)
4. **声明的产品符合的欧盟法规**:列出所有适用的指令 / 法规
   - 例如:EMC Directive 2014/30/EU + LVD Directive 2014/35/EU + RoHS Directive 2011/65/EU + RED Directive 2014/53/EU
5. **采用的协调标准**:列出测试依据的标准号
   - 例如:EN 55032:2015(EMC 发射)+ EN 55035:2017(EMC 抗扰度)+ EN IEC 62368-1:2020(音视频 / IT 设备安全)
6. **附加信息**:测试报告编号、欧代信息、签字人姓名 + 职位
7. **声明日期 + 签字**

**DOC 模板**网上能搜到很多(欧盟官方也有标准模板),**每个产品根据指令不同要单独写一份**,**不要全公司共用一份**。

> DOC 必须**保存 10 年**(欧盟指令 768/2008/EC 规定),建议存在云端 + 纸质副本。

## 你要做什么(按你的情况)

### 情况 A:你卖电子 / 玩具 / 机械(强 CE 类)

1. **先做产品测试**:找 SGS / TÜV / Intertek / Bureau Veritas,做**对应指令的合规测试**,拿到测试报告(约 2000-15000 欧元 / 指令)
2. **基于测试报告写 DOC**(每个产品一份,签字)
3. **下载 CE 矢量图**(SVG),交给印刷厂 / 包装供应商,要求打 5mm 以上
4. **亚马逊后台**:在产品 listing 上传"产品图片(含 CE 标志)"+"包装图片"+"DOC 扫描件"
5. **欧代存档**:把 CE 测试报告 + DOC 副本发给欧代,存档 10 年

### 情况 B:你卖服装 / 家居 / 装饰品(无电无机械)

大部分情况下**不需要 CE**(服装归 REACH 化学品法规,装饰品多数豁免);**但要打 CE 标志的代工厂 / 上游供应商有时会主动给**,**可以打**。**注意**:不带电产品,CE 标志是**可选的**,不强制;但一旦打了,**DOC 也要跟上**。

### 情况 C:你卖二手 / 古董

**CE 标志豁免**(前提是产品原始投放市场时就有 CE),但**DOC 一般找不到了**(都是几年前的文件)。**实务上**:亚马逊会要求你提供"前所有人合规的证明",**这是另一个故事** —— 建议直接找合规顾问。

## 常见问题

**Q1:亚马逊后台没要求我上传 CE 文件,我还要做吗?**
A:**要**。亚马逊后台**不是"你上传了我才要"**,是"你必须随时能拿出来"。市场监管 / 海关抽检时,你拿不出 CE 报告 + DOC,**listing 立刻下架 + 账号可能受影响**。

**Q2:CE 标志可以用中文写"欧盟合格"代替吗?**
A:**不行**。CE 是**唯一被欧盟法规认可的标志**,文字表述(中 / 英 / 德 / 法)都不能替代字母标志。**CE 必须以字母形式出现**。

**Q3:CE 标志能不能和品牌 logo 一起打?**
A:可以,但**CE 标志必须保持完整**。比如把 CE 嵌进 logo、或者把 CE 放在 logo 旁边,**只要 CE 标志本身比例不变、不被遮挡就行**。

**Q4:我打 CE 标志,但产品其实没过测试,会被发现吗?**
A:**很可能**。亚马逊对"高风险类目"(电子、儿童、医疗)有**主动审核**;欧盟海关 2024-2025 年加强了对中国进口商品的合规抽查;**市场监管机构每年 3.15 类似的"打假季"会集中查 CE**。一旦查到,**罚款 + 召回 + 账号关停**。

**Q5:CE 标志可以用红色 / 彩色吗?**
A:可以,**任意颜色都可以**,只要和背景对比度够。实际中**90% 都是黑色 / 深灰色**(通用且专业)。

**Q6:我用的 OEM 代工厂,他说"有 CE 标志"我就能直接用吗?**
A:**要看 OEM 工厂给的资料**:
1. **CE 测试报告**:报告上的"申请人 / 制造商"是谁?如果是 OEM 工厂自己,你要让工厂**把申请人改成你公司**或者**加你的公司名**(否则 CE 证明的不是你的产品)。
2. **DOC**:DOC 签字人必须是"对产品负责的人"(**品牌方或制造商**),OEM 工厂不能单独签。
3. **欧代**:欧代协议**以你的公司名义签**(OEM 工厂可以代付费用,但主体是你)。
4. **CE 标志归属**:CE 是**自我声明** —— OEM 工厂打 CE,**法律责任在工厂**;你要打 CE,**法律责任在你**。**建议**:OEM 工厂 + 你公司**联合声明**(DOC 上写两个主体),并让 OEM 提供测试报告给你。
