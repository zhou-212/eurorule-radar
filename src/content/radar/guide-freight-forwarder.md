## 一句话结论

**头程 = 中国到亚马逊海外仓(FBA 仓)的那段国际物流**。**新手第一次走 100kg 左右,直接空运试水,稳定后转海运散货(LCL),月销 200+ 件再考虑海运整柜(FCL)**。**选货代看 3 件事:① 是不是 FBA 专线(直送 FBA 仓,不是送到你海外地址) ② 是不是双清包税(包中国出口 + 欧盟进口 + 关税) ③ 时效能不能写进合同(敢写"15 天到仓"就靠谱,含糊其辞的别选**。

## 这是啥

头程的"头"= 起点的意思,就是"从工厂出门到亚马逊 FBA 仓上货架"这一整段路。

它包含 3 段:

- **中国端**:工厂提货 → 中国报关出港(出口清关)→ 装船/装机
- **国际段**:海运 / 空运 / 快递跨境运输
- **欧盟端**:到港清关(进口清关 + 缴关税 VAT)→ 提货 → 送到 FBA 仓 → 亚马逊收货上架

**绝大多数新手没意识到**:**这 3 段是 3 个独立的"责任人"**。

- 中国报关 → 货代帮你搞(中国出口商提供装箱单 + 商业发票)
- 国际运输 → 货代包(船公司 / 航空公司)
- 欧盟清关 → 货代 + 你的 EORI 号 + 你的 VAT 号 + 欧代(如果产品需要)
- FBA 入仓 → 货代派卡车送 FBA 仓(预约 + 等待卸货)

**这中间任何一段卡住,你的货都到不了 FBA 仓** —— 比如货代没帮你搞定欧盟清关 = 货被扣在鹿特丹港,每天 50-200 欧的仓储费;或者货代没 FBA 仓预约 = 卡车到了 FBA 门口被赶回来,二次预约费 100-300 欧。

> 打个比方:头程像"国际搬家",工厂是出发地,FBA 仓是目的地,货代是"搬家公司 + 海运 + 海关代办"打包服务。**搬家公司的水平直接决定你货到仓是 15 天还是 60 天,以及中间要不要多交 2000 欧"意外费"**。

## 为什么跟你有关

头程选错,3 个直接后果:

1. **货没到仓 = 没货卖**:头程时效失控(说 30 天实际 60 天),你新店铺还没开张,listing 已经在那里空着等货,广告烧了一周没东西卖,白扔
2. **资金链断裂**:头程费用是"先付钱后到货",空运 100kg ≈ 8000-15000 元,海运整柜 ≈ 30000-80000 元;**如果货代跑路或货被海关扣了,这笔钱直接蒸发**
3. **账号合规风险**:头程的"清关文件"和"欧代信息"是亚马逊反查重点;**清关文件造假(比如用别人的 VAT 号)→ 亚马逊审核打回 + 海关罚款 + 账号"Account Health"标红**

**新手第 1 票头程是"试金石"** —— 走通了,后面加量就行;走不通,亚马逊欧洲站基本就没戏(等你搞明白头程,半年过去了)。

## 4 种头程方式对比表

| 方式 | 适合货量 | 时效(中国到 FBA 仓) | 单 kg 成本(参考) | 优势 | 劣势 |
|------|----------|----------------------|------------------|------|------|
| **国际快递(DHL/FedEx/UPS)** | 0-100kg(试款期) | 5-10 天 | 30-60 元/kg | 最快;门到门;无最小量 | 最贵;电池液体受限;清关自负责 |
| **空运(传统空运 + 报关)** | 100-500kg(中批量) | 7-15 天 | 18-30 元/kg | 中等价格;中等时效 | 要双清;到机场不送到 FBA 仓(末端再加卡车) |
| **海运散货 LCL(Less than Container Load)** | 100-2000kg(中大批量) | 25-40 天 | 4-8 元/kg | 单价最便宜;FBA 专线直送;双清包税成熟 | 慢;有拼箱等货时间;港口间拼箱有损耗风险 |
| **海运整柜 FCL(Full Container Load)** | 2000kg+(20 尺柜 ≈ 28000kg) | 30-45 天 | 2-4 元/kg | 单位成本最低;无拼箱风险;适合稳定出单产品 | 货量门槛高;整柜需 1-2 个月销售才能清掉 |

**关键时点**:

- **Q4(10-12 月)**:海运时效会拉长 5-10 天(圣诞前补货需求暴涨,船公司爆舱);FBA 仓爆仓,预约入仓时间多 1-2 周
- **春节(2 月)**:中国工厂放假,出货慢 1-2 周;节后船公司爆仓
- **平时**:3-9 月是头程淡季,海运时效稳定 25-35 天,价格也有议价空间

**怎么选**:**货量 + 时效需求** 两个轴:

- 货量 100kg + 时效紧 → 快递
- 货量 100-500kg + 试运营 → 空运
- 货量 500-2000kg + 稳定运营 → 海运散货
- 货量 2000kg+ + 主力产品 → 海运整柜

## 选货代的 5 步流程

### 步骤 1:列清楚你的需求(别上来就问"多少钱")

货代报价的前提是知道你要什么。**先把以下信息整理成 1 张表**:

- 货量(kg)和体积(m³)
- 产品类目(普货 / 带电 / 液体 / 危险品)
- 目的 FBA 仓国别(德国 DE / 英国 UK / 美国 US / 日本 JP)
- 时效要求(几号到仓)
- 是不是要双清包税
- 是不是要 FBA 仓直送(预约 + 末端卡车)

**没列清楚就开始问"多少 1 kg",你会被 30 个货代问 30 次同样问题,自己也累**。

### 步骤 2:问 3 个核心问题筛掉 80% 的货代

1. **"能不能做 FBA 专线,送到 [具体 FBA 仓代码,比如 DE1/DE2]?"**
   - 能 → 进入候选
   - 不能 / 含糊 → 砍掉
2. **"是不是双清包税?VAT 和 EORI 你们怎么处理?谁来出?"**
   - 双清包税 + 你只需要提供 VAT + EORI 号 → 靠谱
   - 说"你们自己提供" / "我帮你借一个" → 砍掉(借号清关是违规)
3. **"时效能不能写进合同?晚到一天怎么赔?"**
   - 敢写"25-30 天到 FBA 仓,超过 35 天按 XX 比例赔" → 靠谱
   - "我们时效很稳定" / "一般不会晚" → 砍掉(没约束 = 出了事不管你)

### 步骤 3:问 3 家以上比价(别只问 1 家)

**货代行业水分极大**。同一个柜子,货代 A 报 38000,货代 B 报 52000,差 35% 是常事。

- 问 3-5 家:阿里国际站、百度搜索、亚马逊卖家群(微信群 / 钉钉群 / 论坛)
- 比的维度:**总价 + 时效 + 双清包税范围 + FBA 仓直送 + 是否含保险**
- 选中间价或中间价偏上的(最低价的 90% 是坑:加项费 / 隐含费 / 时效虚标)

### 步骤 4:第一次小批量试水(50-100kg)

**别一上来就走 1 整柜**。新货代先走 50-100kg 试水,验证:

- 时效:说 15 天到仓,实际多少天
- 清关:有没有让你补材料(欧代证书 / 检测报告)
- 末端:卡车送到 FBA 仓有没有问题(预约 / 等候 / 拒收)
- 客服:出问题能不能找到人(响应速度)

**试水没问题,再加量**。**试水出问题,马上换货代** —— 别"再给他一次机会"。

### 步骤 5:签合同 / 留证据(走货必留单据)

**每次走货,以下单据要留底 5 年**:

- 货代合同(报价单 + 服务范围 + 时效承诺 + 赔付条款)
- 装箱单 + 商业发票(出口清关用)
- 提单 / 运单(海运 B/L / 空运 AWB)
- 报关单 / 清关单(中国出口 + 欧盟进口)
- VAT + EORI 号使用记录(清关代理用了你的号,亚马逊后台能查)
- FBA 入仓预约单 + 亚马逊收货回执

**这些单据是"出问题时唯一的证据"** —— 货丢了 / 被海关扣了 / FBA 拒收,没有这些 = 钱货两空。

## 头程前的 3 件必备(没准备好别发)

### 1. VAT 号(增值税号)

- 欧盟进口必备,在任一欧盟国家申请(德国 DE、法国 FR、意大利 IT、西班牙 ES、波兰 PL 等)
- 申请周期:德国约 4-8 周(其他国 1-4 周)
- 申请时需要:**公司营业执照 + 法人护照 + 欧盟当地税务代表(税务代理)**
- 没 VAT = 货到欧盟清关时被卡(海关要求先有 VAT 才能进口商业货物)

### 2. EORI 号(经济运营商注册识别号)

- 欧盟进口必备,在申请 VAT 的国家同步申请(通常免费,1 周拿号)
- EORI 格式:DE + VAT 号(德国),FR + VAT 号(法国),以此类推
- 没 EORI = 货物无法清关

### 3. 欧代 + 包装合规

- 涉及"安全类"产品(电子、玩具、化妆品、食品接触材料等)需要**欧代**(欧盟授权代表)
- 包装上要印:**欧代公司名 + 邮政地址 + 联系方式**(具体格式看产品类目)
- 头程的清关文件 / FBA 入仓时亚马逊抽查包装 → 欧代信息印错了 / 没印 = 扣货 + 拒收

> 关联:[欧盟 GPSR 通用产品安全法规](/eu/radar/eu-gpsr-responsible-person) —— 欧代 + 包装合规必读

## 常见误区(新手 100% 会踩)

### 误区 1:"哪家便宜选哪家"

**最常见的错误**。头程不是机票(越便宜越好),是"按时把货送到 FBA 仓"的服务。

- 便宜货代常有的坑:
  - 报价"裸价"不含双清包税(到港再加 1500-3000 欧)
  - 时效虚标(说 15 天实际 35 天)
  - 没 FBA 仓预约(到门口被赶回来,二次预约费 200-500 欧)
  - 末端卡车甩货(货卸在港口仓库不管了,每天仓储费 50-200 欧)

**怎么避**:**总价对比,不是单价对比**。**总价 = 货代报价 + 所有加项 + 风险成本**。

### 误区 2:"FBA 头程,我自己搞清关也行"

**理论上可以,实际千万别**。欧盟清关是专业活:

- HS Code 编码(海关税则号)错了 → 税率算错,要么多交要么被罚款
- 商品价值申报不实 → 海关抽查时按"申报价 + 罚款"处理
- 文件不齐(CE 证书 / 检测报告 / 欧代协议) → 货物滞留 7-30 天

**新手第一次走货,清关让货代做**。**等你月均 5 柜以上 + 找到了固定的清关代理,再考虑自己清关**。

### 误区 3:"等货到了 FBA 仓再处理 VAT/EORI"

**这是新手最致命的错误**。**货到港发现没 VAT/EORI = 货被扣在海关**,每天罚款 + 仓储费 + 滞期费。**头程没发出去之前,VAT/EORI 必须已经到手**。

**正确节奏**:

- 入驻亚马逊欧洲站第 1 周:申请 VAT + EORI
- 等 4-8 周:号到手
- 第 5-6 周:发头程(等号期间可以选品 / 备货 / 准备 listing)

### 误区 4:"海运便宜,一上来就走海运整柜"

海运整柜 ≈ 30000-80000 元 + 30-45 天。**新手月销都没测出来,1 整柜的货卖 6 个月都卖不完** → 6 个月后亚马逊收你超龄库存附加费,可能比头程费还贵。

**正确节奏**:

- 第 1 票:100kg 快递(8000-15000 元)→ 试款期,验证产品能不能卖
- 第 2-3 票:100-500kg 空运(5000-15000 元)→ 验证稳定单量
- 第 4 票起:500-2000kg 海运散货(5000-20000 元) → 主力货量
- 月销稳定 2000+ 件 + 多个 SKU:1 整柜海运 → 压成本到极限

### 误区 5:"用别人的 VAT / EORI 号清关"

**这是红线**。**借号清关 = 走私**,被查到:

- 货没收 + 罚款(欧盟海关按"申报价值 × 0.5-3 倍"罚款)
- VAT 号持有人连带(被吊销 VAT 资格 + 5 年内不能再注册)
- 亚马逊账号"Account Health"标红(海关数据互通,亚马逊能查到)
- 严重:法人被欧盟海关系统标记,后续无法入境

**怎么避**:**必须自己申请 VAT / EORI,不能用别人的**。

## 何时走头程(产品 / 时机)

### 适合走头程的情况

- ✅ **FBA 卖家**:FBA 模式必须头程到 FBA 仓
- ✅ **产品验证通过**:listing 上线后前 50-100 单,客户反馈 OK
- ✅ **稳定出单 ≥ 30 件/月**:有持续订单才值得走头程
- ✅ **资金链 OK**:头程 + FBA 仓储 + 广告,3 个月回本预算
- ✅ **产品合规完成**:CE / FCC / GPSR / 欧代 / 包装合规都已到位

### 不适合走头程的情况

- ❌ **产品还在选品阶段**:连 listing 都没上,先别发头程
- ❌ **VAT / EORI / 欧代 没到位**:**头程前必须都有**(等 4-8 周拿号期间先做其他事)
- ❌ **资金紧张**:头程 + FBA 仓储费 = 3-6 万人民币,资金不够别硬撑
- ❌ **产品有"季节性"且在季末**:圣诞/复活节后,下一季还早,别发整柜压库存
- ❌ **危险品没确认**:带电 / 液体 / 粉末 / 强磁,先问工厂 MSDS / UN38.3 报告,确认能发再走

## FAQ

**Q1:头程费用一般占总成本的多少?**
A:**8-20%**。以海运散货为例,500kg 货 + 60 元/kg 单价 = 30000 元头程;如果产品售价 100 元,头程占比 6-8%;如果售价 30 元,头程占比 15-20%。**售价越低的产品,头程越敏感**。新手尽量选"客单价 100 元以上"的产品,头程压力小。

**Q2:海运散货和海运整柜,除了货量区别,还有什么不一样?**
A:1) 时效:FCL 直发 30-35 天到仓,LCL 要等"拼满 1 柜"才能发(等 1-2 周),总时效 35-50 天。2) 风险:LCL 是"和别人的货拼一个柜",出现破损 / 错货 / 海关扣货,**分不清是哪个货主的货**,理赔难;FCL 全是你自己的货,责任清晰。3) 价格:单 kg 成本 LCL 比 FCL 贵 30-50%(船公司按"柜"卖,平摊到每 kg)。

**Q3:头程保险要不要买?**
A:**强烈建议买**。保费一般是货值的 0.1-0.3%,50000 元的货保 500 元的费。海运丢货 / 破损 / 海关扣货的案例不少,**没保险的货丢了,货代赔不赔看合同,大概率不赔**。

**Q4:空运是不是"贵但快"?能一直走空运吗?**
A:对,空运单 kg 成本是海运的 4-6 倍。**长期走空运不现实**:月销 1000 件,空运费可能吃掉 30-40% 的利润。空运只适合"试款期 / 紧急补货 / 高客单价产品"。**主力产品稳定后必须转海运**。

**Q5:货代说"包关税",是不是双清包税?**
A:**"包关税"和"双清包税"是两件事**。双清包税 = 中国出口清关 + 欧盟进口清关 + 关税 + 增值税全部包。"包关税"可能只包关税,不含清关服务费 / 仓储费 / 派送费。**签合同时要看清包含范围**。问"是不是双清包税?有没有额外费用?"。标准报价应包含:① 提货 ② 中国出口报关 ③ 海运 / 空运 ④ 欧盟进口清关 + 关税 + VAT ⑤ 末端派送到 FBA 仓 ⑥ FBA 仓预约。

**Q6:FBA 仓拒收怎么办?**
A:常见原因:1) 包装标签错误(FNSKU / 亚马逊外箱标签错);2) 超尺寸 / 超重量;3) 危险品没申报;4) 包装破损。**拒收后货代会拉回仓库**,你需要:重新贴标 / 重新包装 / 重新预约入仓,每次加 100-500 欧。**预防**:发头程前严格按亚马逊包装指南(https://sellercentral.amazon.com/help/hub/reference/GUDDFBDJU3EYV5NP) 检查。

**Q7:头程时效延误,亚马逊要"缺货罚款"怎么办?**
A:亚马逊对"Listing 缺货超 7 天"会降低 search ranking(搜索排名)。**头程延误,listing 一旦断货,恢复要 1-2 个月**(排名要重新爬)。**预防**:**永远保持 30-45 天 FBA 库存**,不要"卖完再补"。**应急**:**保留 1-2 个备选货代,主货代出问题立刻切**。

**Q8:可以一票头程发到多个 FBA 仓吗?**
A:可以,叫"分仓"。**Pan-EU 模式**:亚马逊自动在你开启的国家分配库存(德国 / 法国 / 意大利 / 西班牙),你只发 1 国(比如德国),亚马逊帮你分到 5 国仓(要额外开 Pan-EU)。**多国发货**:你自己让货代一票货分到 5 国 FBA 仓(货代收分仓费,每个分仓点 100-300 欧)。**新手建议**:**前期 1 国(德国)发货,月销稳定 200+ 再开 Pan-EU**。
