## 一句话结论

如果你在 Amazon EU 卖**14 岁以下儿童玩具**(毛绒、塑料、电动、木质、户外、拼图、积木),欧盟玩具安全指令 2009/48/EC 已经强制执行 15 年 —— **必须 CE 标志 + DoC + EN 71 系列协调标准测试**(物理机械 + 易燃性 + 化学 3 大基础 + 19 种迁移元素)。**电动玩具**还要加 EN IEC 62115 标准。包装上**强制印警示语**(年龄警告 / 小部件警告)。亚马逊对儿童玩具强制 5 项合规,违规扣货 + listing 限流。

## 这是啥

玩具安全指令 **Directive 2009/48/EC**,2009-06-18 通过,**2011-07-20 起强制**,**替代**原 **88/378/EEC** 指令。法律性质是 **Directive**(指令),各成员国转化为本国法律后执行(德国玩具法 / 法国 Code de la consommation 玩具章节 / 意大利 Codice del Consumo)。

协调标准是 **EN 71 系列**(欧洲玩具安全标准),由 **CEN**(欧洲标准化委员会) / **CENELEC**(欧洲电工标准化委员会)发布,定期更新。截至 2025 年,EN 71 系列已发布 14+ 项子标准,覆盖各类玩具。

**核心 3 大基础测试**:
- **EN 71-1:2014+A1:2018**:物理机械性能(尖锐边缘、锐利尖端、小部件、断裂、咬合、挤压)
- **EN 71-2:2020**:易燃性(所有玩具)
- **EN 71-3:2019+A1:2021**:**化学 — 19 种迁移元素限值**(最常踩雷的标准,3 类材料:涂层 / 聚合物 / 纸张/纸板)

**其他常见子标准**:
- **EN 71-4:2018**:化学实验套装
- **EN 71-5:2015**:非实验化学玩具(化妆品玩具 / 创意工艺)
- **EN 71-7:2014+A1:2018**:手指画颜料
- **EN 71-8:2018**:户外玩具(秋千、滑梯)
- **EN 71-9 至 12**:有机化合物(单体、增塑剂、防腐剂等)
- **EN 71-13:2021**:嗅觉棋盘游戏
- **EN 71-14:2018**:蹦床
- **EN IEC 62115:2020+A11:2020**:**电动玩具安全**

## 适用 vs 不适用

**适用**(典型,2009/48/EC 范围):

- ✅ 14 岁以下儿童设计或使用的所有玩具
- ✅ 毛绒玩具(娃娃、玩偶、动物玩偶)
- ✅ 塑料玩具(积木、拼图、模型)
- ✅ 电动玩具(遥控车、电动火车、电动动物)
- ✅ 木质玩具(积木、火车、拼图)
- ✅ 户外玩具(秋千、滑梯、跷跷板)
- ✅ 化学 / 工艺玩具(实验套装、橡皮泥、颜料)
- ✅ 拼图 / 棋盘游戏
- ✅ 节日 / 派对玩具(圣诞装饰、化妆服装)
- ✅ 婴儿用品(牙胶、摇铃、布书、安抚奶嘴)
- ✅ 儿童自行车头盔
- ✅ 婴幼儿安全座椅
- ✅ **儿童饰品**(项链、手链、耳环、戒指)
- ✅ **儿童磁铁玩具**
- ✅ **水上充气玩具**

**不适用**(典型豁免):

- ❌ 14 岁以上儿童和成人使用的玩具(如收藏模型、成人拼图)
- ❌ 装饰性节日用品(圣诞树、装饰花环,非玩具)
- ❌ 体育用品(自行车、滑板、滑雪板)
- ❌ 娱乐设施(游乐场固定设备、自动售货机)
- ❌ 视频游戏机 / 软件(另有 RED 指令)
- ❌ 烟花(另有 Pyrotechnic Articles 法规)
- ❌ 食品(另有食品安全法规,但食品玩具混合产品需特殊处理)
- ❌ 化妆品玩具(如有化妆成分,需同时符合化妆品法规 1223/2009)

## 为什么跟你有关

**亚马逊公开规则**(Seller Central 多年要求):
> "All toys sold in the EU targeting children under 14 must comply with Toy Safety Directive 2009/48/EC. Sellers must provide CE Mark, DoC, EN 71 test reports, and age warnings via Seller Central."

**没做的后果**(2024-2025 公开案例):

- **2024-08 案例**:亚马逊 / TEMU 联合中国海关,扣押 17 万件不合规儿童玩具(主要是电动玩具未做 EN 62115 + 邻苯超标)
- **2024-12 案例**:德国市场监管部门对 4000+ 跨境直邮儿童玩具 listing 限流(EN 71-3 镉、铅、镍释放超标)
- **2025-Q1 案例**:法国 DGCCRF 对儿童饰品(项链、手链)开展镍释放 + 镉专项抽查,3500+ listing 被要求 30 天内补 EN 71-3 测试报告
- **2025-Q2 案例**:意大利 MOH(卫生部)对儿童化妆玩具(儿童彩妆、儿童香水)开展专项执法,900+ listing 下架

中国卖家最常见的踩雷点:
- **软塑料玩具 / 牙胶 / 摇铃**:邻苯二甲酸酯(DEHP/DBP/BBP/DIBP) + 镉 + 铅
- **儿童饰品**(项链、耳环、手链):**镍释放**(EN 71-3) + 镉 + 铅
- **毛绒玩具**:涂层铅、染色偶氮染料(EN 71-9 ~ 12)
- **木质玩具 / 拼图**:涂料重金属 + 木材防腐剂
- **电动玩具**:EN 62115 电池安全 + 机械危险 + 邻苯
- **儿童彩妆 / 化妆玩具**:除了 EN 71,还要符合化妆品法规 1223/2009
- **学步车 / 婴儿座椅**:坠落风险 + 警告语

## 5 步行动框架(总预算 3000-10000 €)

### 第 1 步:产品类目 + 年龄段确认(预算 0 €,1 周)

**做什么**:
- 列出所有投放欧盟市场的儿童产品,按**年龄分段**归类:
  - **0-3 岁(婴儿)**:最严,小部件禁令、窒息风险
  - **3-6 岁(学龄前)**:高风险,小部件 + 化学 + 物理
  - **6-14 岁(学龄)**:较宽,主要化学 + 物理
- **每款产品做"玩具属性清单"**:SKU / 类目 / 适用年龄 / 材质 / 是否有电动部件 / 是否有磁铁 / 是否有化学成分 / 是否有小部件
- 区分**玩具 vs 儿童用品** vs **儿童饰品** vs **儿童电动玩具** —— 各自测试组合不同

**为什么要做这一步**:
- 玩具测试按"年龄 + 类目"计费,不同年龄 / 类型 → 测试项目组合不同
- 拆解后可以识别"哪些产品需要 EN 71 系列哪几项" → 节省 30-50% 成本
- 拆解过程会发现"哪些是高风险"(婴儿用品、电动玩具、化学玩具) → 优先做

**避坑**:
- 婴儿用品(0-3 岁)和小部件禁令是 EN 71-1 的**最严段** —— 任何部件可放入小部件圆筒(直径 31.7 mm × 长 57.1 mm) = 不合规
- 儿童饰品(项链、耳环) — EN 71-3 镍释放和镉限值最严
- 电动玩具 — 还要 EN IEC 62115 测试,不能省
- 节日 / 派对玩具(圣诞装饰、化妆服装) — 也是玩具,需要 EN 71

### 第 2 步:EN 71 系列测试(预算 2500-8000 €,3-6 周)

**必做项**(基础 3 项 + 类目专属):
- **EN 71-1:2014+A1:2018** 物理机械(所有玩具)
- **EN 71-2:2020** 易燃性(所有玩具)
- **EN 71-3:2019+A1:2021** 19 种迁移元素(所有玩具)
- **EN 62115:2020+A11:2020** 电动玩具(电池 / USB 供电)
- **EN 71-4** 化学实验套装(若适用)
- **EN 71-5** 化妆品玩具(若适用)
- **EN 71-7** 手指画颜料(若适用)
- **EN 71-8** 户外玩具(秋千、滑梯)
- **EN 71-9 ~ 12** 有机化合物(EN 71-9 是最常被引用)

**找谁做**:
- Notified Body(测试机构):SGS / TÜV SÜD / TÜV Rheinland / Intertek / Bureau Veritas / CTI 华测 / Eurofins
- 选有 EN 71 系列直接出报告资质的(报告上含 NB 编号)
- 报价区间:基础 3 项单 SKU 2500-4500 €,电动玩具 + 2000-3000 €

**关键**:
- 检测报告**保留 10 年**(CE 标志相关文档要求)
- 报告要**含品牌名 + SKU + 检测日期 + 检测机构印章 + NB 编号**
- **跨 SKU 相似产品可以合并检测**(节省 30-50% 成本)
- **涂层 / 聚合物 / 纸张**3 类材料**每类都要测**
- 婴儿用品(0-3 岁)**加严小部件测试** — 报告里要明确注明

### 第 3 步:DoC 起草 + 签发(预算 0-800 €,1 周)

**DoC 必备内容**(基于 2009/48/EC + 协调标准 EN ISO/IEC 17050):
- 制造商 / 欧盟授权代表(Authorised Representative)信息
- 产品名 / 型号 / SKU
- **适用法规清单**:2009/48/EC + 协调标准(EN 71 系列对应项)
- 检测报告引用(机构 + 报告号 + 日期)
- 适用年龄 / 安全警告
- 安全评估摘要(基于协调标准)
- 签发日期 + 签发人签名(姓名 + 职位)
- 制造商 / 授权代表盖章

**自起草 vs 委托**:
- 简单 SKU(单一类目、单一 SKU):自起草(用 EN ISO/IEC 17050 模板)
- 复杂 SKU(多类目、多 SKU、电动玩具):委托合规顾问(500-800 € / SKU)
- 模板可参考:TÜV / SGS 官网提供的玩具 DoC 模板

### 第 4 步:标签 + 包装 + 警告语(预算 500-1500 €,1-2 周)

**必印内容**:
- **CE 标志**(玩具强制要求)
- **生产者 / 进口商名称 + 欧盟地址**(包装上)
- **型号 / 批号 / 序列号**(追溯用)
- **警告语**(强制,具体格式见下)
- **多语种说明书**:至少 EN + DE + FR + IT + ES 5 国语言

**警告语要求**(2009/48/EC + EN 71 系列):
- **年龄警告**:适用年龄 / 不适用年龄
- **小部件警告**:3 岁以下儿童不适用
- **特定危险警告**:如"含磁铁,吞食有危险"、"含纽扣电池"等
- **使用说明**:必要的注意事项

**典型警告语格式**(玩具安全指令 Annex V):
```
警告:不适合 3 岁以下儿童使用 — 含小部件
WARNING: Not suitable for children under 3 years of age — contains small parts
```

**避坑**:
- 警告语必须有 + 用销售国语言(亚马逊 DE / FR / IT / ES 至少各自国语言)
- 印在产品本体或包装上,字高 ≥ 1mm(或满足 EN 71-1 视距要求)
- **小部件警告 + 年龄警告是双层结构**,不能只印一个
- 电动玩具 + 化学玩具额外警告必须印

### 第 5 步:亚马逊后台 + 详情页同步(预算 0 €,1 周)

**后台字段**(Seller Central):
- `Manage Inventory → Compliance → Toy Safety`
- 每个 SKU 填:类目 / 适用年龄 / 警告语内容 / 检测报告号
- 上传:DoC 扫描件 + EN 71 检测报告 PDF

**详情页更新**:
- `Compliance` 板块加:玩具安全指令合规说明 + 多语种版本
- 主图区域可加"CE Marked"或"EN 71 Tested"标识(亚马逊官方允许)
- 关键文档:DoC PDF + EN 71 测试报告(可在详情页 Compliance 板块提供下载)

**亚马逊抽查流程**:
- 平台**批量合规团队**按 ASIN 抽查,首次未通过发警告邮件
- 二次未通过 → 限流 / listing 在搜索结果中降权
- 三次未通过 → 强制下架 + 资金冻结 90 天
- 跨境直邮小包 → 海关直接扣货(尤其节日玩具旺季)

## 4 个常见误区(基于公开报告)

**误区 1:"我的玩具看起来很安全,不用做 EN 71"**
错。EN 71-3 迁移元素是**化学测试**,看不见摸不着。**塑料玩具的邻苯 / 镉 / 镍释放,外观看不出来**。外观"看起来安全" ≠ 化学"实际安全"。2024 年德国下架的 4000+ listing 中,**多数是"看起来正常"的儿童玩具**。

**误区 2:"我做了 EN 71-1 物理测试,就够了"**
错。EN 71-1 物理 + EN 71-2 易燃 + EN 71-3 化学 = **基础 3 项**,缺一不可。**只做物理 = 0 化学合规**。化学测试是 EN 71-3,2024-2025 下架案例中,化学物质超标占 70%+。

**误区 3:"儿童饰品不是玩具,不需要 EN 71"**
错。**14 岁以下儿童设计 / 使用的物品 = 玩具**,包括**儿童饰品**(项链、手链、耳环)。**镍释放 + 镉 + 铅**是 2024-2025 法国 / 德国执法的重灾区,**法检率 100%**(金属类必检)。

**误区 4:"CE 标志印在产品上 = 合规"**
**CE 标志本身不是认证,是声明**;CE 标志的支撑文件是 DoC + EN 71 测试报告。**有 CE 标志但没 DoC / 没测试报告 = 违规**(欺诈性 CE 标志,处罚更严)。

## 何时做 / 何时不做

**符合以下任一条件,立刻做**:
- ✅ 你在 Amazon EU 销售 14 岁以下儿童玩具(任何类目)
- ✅ 你 2026 年后还要继续卖
- ✅ 你的产品是儿童饰品 / 儿童电动玩具 / 儿童化妆玩具
- ✅ 你重视 listing 长期稳定性

**暂缓**:
- ⚠️ 你 2025-2026 年内准备退出欧洲市场
- ⚠️ 你的产品明确标注 14 岁以上使用(且亚马逊后台有"adults only"勾选)
- ⚠️ 你的产品是纯非玩具(运动器材、乐器、专业设备)

## 如果你只做一件事

**第一步先做"玩具属性清单 + 年龄段"** —— 把所有产品按年龄分段(0-3 / 3-6 / 6-14),识别"哪些是婴儿用品(0-3 岁最严)"、"哪些是电动玩具(EN 62115)"、"哪些是化学玩具(EN 71-4/-5/-7)"。这一步**不花钱,1 周搞定**,但能让你后续 EN 71 测试预算节省 30-50%,也避免漏检导致全批次扣货。**没有这张表,后面测试就是赌博**。
