## 一句话结论

**亚马逊广告有 5 类硬禁用词**:绝对化用词(最佳 / 最便宜 / 100%)、医疗效果(治愈 / 治疗 / 预防)、比较词(比 XX 更好)、主观描述(最受欢迎 / 销量第一,无证据)、引诱点击(点击查看 / 点这里)。**违反直接拒登,严重的 listing 下架、账户警告**。欧盟额外盯"免费必须明示条件"、价格比较必须准确、促销期限必须真实。**提交前花 5 分钟自查,能挡掉 90% 的拒登**。

## 这是啥

亚马逊广告(Amazon Ads)对"广告文案"有一套**独立于产品 listing 的合规审查系统**。这套审查比 listing 本身的合规还要严,理由很简单:广告是"亚马逊主动推"给买家的内容,出问题亚马逊自己要背锅。

**简单说,广告文案 = 买家直接看到的"销售话术"**:

- 标题(Sponsored Products 的 Headline)
- 五点描述里投放的关键词
- 自定义图片(如果你用 Sponsored Brands)
- 站内站外跳转链接

**这些内容里有 5 类词,亚马逊直接拒登,系统自动识别**。**不靠人工审,靠关键词 + 图像识别 + AI 模型**,所以别存侥幸心理。

更狠的是:**有些词 listing 本身能用,广告里不能用**(比如 "best" 在 bullet points 里有时能过,在广告标题里直接 404)。**这意味着你得"为广告单独写一套文案"**。

## 为什么跟你有关

广告是新手"破冰"流量的工具,但写错词的代价是连锁反应:

1. **广告拒登,listing 流量断档** —— 拒登不是"明天再投",是"该关键词永久不能投"或"该广告活动暂停 7-30 天"
2. **账户警告累计 → 封号** —— 亚马逊对"广告违规"是"累犯加重"制,第一次警告,第二次限制功能,第三次直接暂停广告账户(整个店铺没法投广告)
3. **listing 下架 → 之前的广告投入打水漂** —— 严重违规(医疗效果、虚假宣传)会触发 listing 整体下架
4. **新手 99% 不自查** —— 多数新卖家"凭感觉写广告词",被拒了才回去查,**时间 + 预算双重浪费**

**最讽刺的是**:亚马逊官方文档把禁用词列得很细,但新手几乎没人去读,**等被拒了再翻文档 = 已经亏了几十欧广告费**。

**广告合规 = 卖家最便宜的"保险"**。花 5 分钟自查,挡掉几千欧的损失。

## 5 类硬禁用词速查表

| 类别 | 典型词 | 为什么禁用 | 改写方式 |
|------|--------|------------|----------|
| **绝对化用词** | 最佳 / 最便宜 / 最优 / 第一 / 100% / 绝对保证 / 顶级 / 完美 | 欧盟《不公平商业行为指令》明令禁止无证据的绝对化表述 | "适合日常使用" / "高性价比" / "实测表现稳定" |
| **医疗效果** | 治愈 / 治疗 / 预防疾病 / 增强免疫力 / 排毒 / 抗癌 / 减肥特效 | 医疗器械 / 药品监管,亚马逊对"擦边医疗效果"零容忍(哪怕你是卖枕头) | "辅助睡眠" / "支撑腰部" / "使用舒适" |
| **比较词** | 比 XX 更好 / 比 XX 便宜 / 优于竞品 / 第一名 / #1 | 主观比较无标准 + 涉嫌诋毁竞品 | 直接描述自己的产品参数(厚度、材质、续航) |
| **主观描述** | 最受欢迎 / 销量第一 / 好评如潮 / 火爆全网 / 网红推荐 | 没有证据 + 涉嫌虚假宣传 | 写"4.5 星评分(基于 200 条评价)" / "X 国畅销" 附数据来源 |
| **引诱点击** | 点击查看 / 点这里 / 更多惊喜 / 戳我 / 必看 | 操纵用户行为,违反亚马逊广告创意规范 | 改成具体的卖点词 / 直接写产品核心功能 |

**特别提醒**:
- **"免费"在欧盟是高危词**(详见下文)
- **"限时"类词**(最后 3 小时 / 错过等一年)要附真实截止时间
- **数字 + 比较**(销量第一 / 好评率第一)**必须有第三方数据源**(亚马逊 Best Seller 标志除外)

## 欧盟特别盯的 3 类词(比美国站更严)

### 1. "免费"必须明示条件

**欧盟《不公平商业行为指令》第 6 条**:广告中说"免费",必须满足:

- 消费者**无需支付任何额外费用**(包括运费、配件费)
- 不能附带"必须买 XXX 才送"的隐性条件
- 不能用"免费"作诱饵但实际收更高价格(原价虚标)

**典型违规话术**:

- ❌ "买 1 送 1 免费"——但运费收 9.9 欧
- ❌ "免费试用 30 天"——但到期自动续费要收 99 欧
- ❌ "免费配件"——但必须买主件才送

**合规话术**:

- ✅ "购买本产品,获赠同款(免运费)" —— 显式说"免运费"
- ✅ "30 天试用,无理由退货运费由我方承担" —— 显式说"运费谁出"

### 2. 价格比较必须真实

**不能虚构"原价 / 划线价"**:

- 划线价必须是**过去 30 天内**的真实最低价(欧盟规定,不同国家有差异)
- 不能"先涨价再打折"(德国 2023 年起重罚)
- "X% OFF"必须显示折扣前 / 折扣后两个数字

**举例**:

- ❌ "原价 99 欧 → 39 欧(6 折)" —— 但你从来没卖过 99 欧
- ✅ "建议零售价 99 欧,促销价 39 欧" —— 有厂家建议价支撑

**特别注意**:德国对"虚假折扣"是刑事犯罪,罚款可达 5 万欧 + 监禁。

### 3. 促销期限必须真实

- ❌ "最后 3 小时" —— 倒计时 0 后还在投
- ❌ "仅此一天" —— 第二天还在
- ❌ "限时特惠" —— 持续 3 个月

亚马逊广告系统会**实时扫描倒计时和实际投放时间**,发现倒计时结束后继续展示,直接拒登。

**合规做法**:促销期真结束就**关掉广告活动**,别让系统倒计时过了广告还活着。

## 站外引流禁忌(违反直接封号)

**亚马逊广告最严的一条:不允许在广告里引导到亚马逊外网站**。

**违规行为**:

- 广告落地页跳到独立站
- 图片里放 WhatsApp / 微信 / Instagram 链接
- "扫码加我领优惠券" / "联系我们,专属折扣"
- 在产品主图加店铺官网 URL

**为什么这么严**:亚马逊花了 20 年建立的买家"在亚马逊闭环下单"心智,卖家往站外引流 = 砸亚马逊饭碗。**这是平台底线,踩一次警告,两次直接暂停广告账户**。

**正确做法**:广告落地页**永远是亚马逊 listing 或品牌旗舰店**(品牌备案后可用),不要碰任何站外链接。

## 3 步提交前自查法(5 分钟搞定)

### 第 1 步:用"5 类禁用词速查表"过一遍

把你要投的广告文案(标题、关键词、图片文字)复制到 Word / Notion,按上面的 5 类表逐个过:

- 看到 "最 / 第一 / 100% / 绝对" → 改
- 看到 "治愈 / 治疗 / 预防" → 改
- 看到 "比 XX 更好" → 改
- 看到 "最受欢迎 / 销量第一" → 必须有证据,没证据改
- 看到 "点击查看 / 点这里" → 改

**这一步 90% 的拒登能挡掉**。

### 第 2 步:欧盟特别词自检

针对"免费 / 价格比较 / 促销期限"三类:

- "免费" → 写明"免运费"或"无附加条件"
- "X% OFF" → 检查原价是否过去 30 天真实最低价
- 倒计时 → 设个日历提醒,倒计时到就关广告

### 第 3 步:用亚马逊官方"广告预览"工具验证

**Amazon Ad Preview**(广告预览):

- Seller Central → Advertising → Campaign Manager → 选你的广告活动 → "Preview"
- 看买家实际看到的广告长啥样
- **图片放大看文字** —— 有些词小到你看不清,但亚马逊系统能 OCR 识别

**另外**:亚马逊有一个**"Advertising Policy"页面**会更新拒登原因:[https://advertising.amazon.com/resources/ad-policy](https://advertising.amazon.com/resources/ad-policy) —— 拒登后第一时间查这个页面,看你具体踩了哪条。

## 常见误区(新手 100% 会踩)

### 误区 1:"我 listing 都过了,广告肯定过"

**错**。listing 合规是 listing 合规,广告合规是广告合规,**两套系统独立审核**。常见情况:

- Listing 里写"最受欢迎",过了(没被比对的精确词触发)
- 同样这句话放到广告标题 → 直接拒登(广告系统更敏感)

**正确认知**:广告文案 = 单独写一套,**别图省事直接复制 listing**。

### 误区 2:"我用英文写,欧盟管不到我"

**错**。亚马逊欧洲站广告政策**统一按最严的德国 / 法国标准**。你在英国站用 "best" 可能没事(英国脱欧后监管略松),到德国站直接被拒。

**正确做法**:欧 5 国(德/法/意/西/荷)的广告文案**按最严标准写**,别赌哪个国家审查松。

### 误区 3:"小语种翻译过来,意思变了,合规问题不存在"

**翻译 ≠ 洗稿**。亚马逊的合规审查是**多语言并行**的:

- 你的英文广告 "Best Quality" 拒登
- 你翻译成德文 "Beste Qualität" 重新提交 → 一样拒登(同一规则)
- 你 "改写" 成 "High Quality" → 英文通过,但德文翻成 "Hochwertige Qualität" 一样可能被卡(德文审查更严)

**正确做法**:**多语言广告都按"欧盟最严标准"写**,别用"小语种漏洞"。

### 误区 4:"被拒了改一下再投,没事"

**错**。亚马逊对"广告拒登"是**累计计算**的:

- 同一广告活动被拒 3 次 → 暂停 7 天
- 同一账户被拒 5 次 → 暂停 30 天
- 同一账户被拒 10 次 → 广告账户永久封禁(无法在所有 ASIN 上投广告)

**正确做法**:**被拒一次就彻底自查,别反复试错**。读官方拒登理由 → 改 → 7 天后再投(让系统"冷却"一下)。

### 误区 5:"自动广告不需要自查文案"

**错**。自动广告(Auto-Targeting)的"文案"是用你 listing 的内容生成的,**listing 里的禁用词一样会触发广告拒登**。**优化 listing = 优化自动广告**。

## 何时做 / 何时不做

### 何时必须做合规自查

- ✅ **新广告活动上线前** —— 必查,5 分钟的事
- ✅ **关键词调整时** —— 新加的词要查
- ✅ **广告被拒登后** —— 100% 要查,查到为止
- ✅ **进入新国家站**(比如从德国扩到法国) —— 不同国家合规标准差异
- ✅ **促销季前**(黑五、网一、圣诞) —— 倒计时 / 折扣类词要严格审

### 何时不用过度担心

- ⚠️ **描述性形容词**(高 / 厚 / 强 / 持久) —— 客观参数,基本都能用
- ⚠️ **产品型号 / 品牌名 / 尺寸数字** —— 不构成宣传,放心写
- ⚠️ **使用场景**(户外 / 办公 / 家庭) —— 客观描述,合规

## FAQ

### Q1:广告被拒登了,多久能重新提交?

**答**:亚马逊没有官方"冷却期",但经验上**至少等 3-7 天再投同一关键词 / 文案**。系统对"短时间内反复提交"会更敏感。**改完文案先自查,再等 3 天重新提交**。

### Q2:我用了 "Best Seller",为什么也被拒?

**答**:"Best Seller" 是亚马逊的注册商标,**只有亚马逊官方颁发的 Best Seller 标志才能用**。你自己写 "Best Seller 2024" 也算侵权(商标 + 虚假宣传)。如果你确实在某子类目是 Best Seller,亚马逊后台会自动加标志,不需要你手写。

### Q3:"100% natural"(100% 天然)能用吗?

**答**:**高危**。"100%" 是绝对化词,亚马逊对"100% 天然""100% 有机"要求**有第三方有机认证(USDA Organic / EU Organic)**。没认证写了"100% 天然" → 拒登 + 营养类目可能下架。**改成"主要成分天然"或"自然成分"** 更稳。

### Q4:医疗效果词只针对"医疗器械"类目吗?

**答**:**错**。亚马逊对"医疗效果"是**全类目零容忍**:

- 卖枕头写"治疗颈椎病" → 拒登
- 卖保健品写"增强免疫力"(免疫类宣称)→ 拒登
- 卖护肤品写"祛痘"(暗示治疗效果)→ 拒登
- 卖减肥产品写"7 天瘦 10 斤"→ 拒登 + 可能下架

**通用规则**:广告文案里**避开所有"治愈 / 治疗 / 预防 / 增强功能"** 类词,改用"辅助 / 适用 / 适用于"。

### Q5:广告被拒登会不会影响 listing 自然排名?

**答**:**不会直接降权,但有间接影响**:

- 广告拒登 → 没有付费流量 → 订单减少 → BSR 下降 → 自然流量减少
- **listing 本身下架**(严重违规时)→ 自然排名直接归零
- **账户警告累计 → 卖家账户绩效分下降** → 搜索权重降低

**结论**:**广告合规问题 = 长期流量问题**,不是"广告拒了就重投"的简单事。
