## 一句话结论

REACH 法规 (EC) 1907/2006 是欧盟的**化学品"宪法"**,管**所有**在欧盟制造或进口的**物质、配制品、物品**(articles) —— 几乎覆盖你 Amazon EU 上架的每一件实物。**两个核心动作**:① **Article 7 通报** —— 产品中 SVHC(高关注物质)≥ 0.1% (w/w)时向 ECHA 通报;② **Annex XVII 合规** —— 数百种禁用 / 限用物质不能含。亚马逊对全品类(玩具 / 服装 / 电子 / 家具 / 珠宝)都有 REACH 合规要求,违规扣货。

## 这是啥

REACH 全称 **Registration, Evaluation, Authorisation and Restriction of Chemicals**(化学品注册、评估、授权和限制),2006-12-18 通过,**2007-06-01 生效**。主管机构是位于芬兰赫尔辛基的 **ECHA**(European Chemicals Agency,欧洲化学品管理局)。

REACH 的革命性在于**把举证责任倒过来** —— 不是政府证明"这化学物质有害",而是企业必须证明"这化学物质安全可用"。每年 ECHA 都会更新两份关键清单:

- **SVHC Candidate List**(高关注物质候选清单):截至 2025 年已超 240 项,每年 1-2 月、7 月各更新一次
- **Annex XVII**(限制物质清单):数百种禁用 / 限用物质 + 具体阈值 + 适用范围

**法律性质**:REACH 是 **Regulation**(法规),直接适用全境 27 个成员国 + 冰岛 / 列支敦士登 / 挪威(EEA),**不需要转法**。从 2007-06-01 至今已运行 19 年,是中国卖家欧盟合规的"必答题"。

## 适用 vs 不适用

**适用**(REACH 几乎覆盖所有实物产品):

- **物质**(substance):单一化学元素或化合物(染料、塑料颗粒、金属粉末)
- **配制品**(preparation/mixture):多种物质的混合物(油漆、胶水、化妆品配方)
- **物品**(article):由一种或多种物质组成、在生产中获得特定形状 / 表面 / 设计的物体(玩具、服装、电子产品、家具、珠宝、塑料袋、**几乎所有实物**)

**不适用**(典型豁免):

- ❌ 放射性物质
- ❌ 海关监管下的过境货物
- ❌ 海关仓储货物
- ❌ 食品和饲料(另有 EU 178/2002 食品法)
- ❌ 化妆品(另有 EU 1223/2009 化妆品法规)
- ❌ 药品(有人用 / 兽用专项法规)
- ❌ 医疗器械的有害物质(另有 MDR)
- ❌ 废物(recycling 后的产品除外)
- ❌ 国防 / 国家安全豁免

## 为什么跟你有关

**ECHA 通报公开数据库**(Article 7 通报,可在 ECHA 官网公开查询):
- 截至 2024 年底,ECHA 累计收到**超 1.2 万件 Article 7 通报**
- 通报量年增长 30%+(2022 年 1800+ 件 → 2024 年 3500+ 件)
- 玩具 / 服装 / 电子 / 家具 / 珠宝是**通报 Top 5 类目**

**亚马逊公开规则**(Seller Central Product Compliance 长期要求):
> "All products sold in the EU must comply with REACH (EC) 1907/2006. Products containing SVHC ≥ 0.1% (w/w) must be reported to ECHA. Sellers must provide SDS (Safety Data Sheet) for chemical products or relevant documentation for articles."

**没做的后果**(2024-2025 公开案例):

- **2024-11 案例**:亚马逊对 8000+ 中国卖家 listing 启动"REACH 物质声明"专项审核,主要集中在服装(偶氮染料)、电子(溴化阻燃剂)、玩具(邻苯二甲酸酯)三个类目
- **2025-Q1 案例**:德国 BAuA(联邦职业安全健康研究所)对亚马逊 / eBay 上的服装 / 配饰类目开展 Annex XVII 偶氮染料专项抽查,1700+ listing 被要求 30 天内补 SDS / 测试报告
- **2025-Q2 案例**:法国 DGCCRF 对珠宝类目开展 REACH 镉 / 铅 / 镍释放专项抽查,2000+ listing 限流

中国卖家最常见的踩雷点:
- **服装 / 纺织品**:偶氮染料(Azo dyes) → 释放致癌芳香胺
- **塑料玩具 / 儿童用品**:邻苯二甲酸酯(DEHP/DBP/BBP/DIBP) + 镉
- **皮革 / 鞋类**:六价铬(Cr(VI)) + 甲醛
- **金属饰品 / 珠宝**:镍释放 + 镉 + 铅
- **电子 / 电器**:溴化阻燃剂(PBB/PBDE) + 铅
- **家具 / 木制品**:甲醛

## 关键阈值 + 清单概览

### SVHC 候选清单(Article 7 通报阈值)

| 维度 | 阈值 |
|---|---|
| SVHC 含量(单物质) | ≥ 0.1% 重量比 (w/w) |
| 通报对象 | ECHA(欧洲化学品管理局) |
| 通报时限 | 投放市场后 6 个月内 |
| 通报内容 | 物质名称 / 含量范围 / 物品类别 / 安全使用说明 |
| 公告对象 | 消费者和下游用户(产品 / 包装 / 网页 45 天内可索取) |

**重点**:0.1% w/w 是按**整个物品**算,不是单一均质材料。例如一个塑料玩具含 0.15% DEHP,无论哪种塑料部件,整件都要通报。

**截至 2025 年,SVHC 候选清单常含物质类别**(不完全列举):
- 邻苯二甲酸酯类(DEHP / DBP / BBP / DIBP / DnHP / DMEP 等)
- 溴化阻燃剂(PBB / PBDE / HBCDD / SCCP)
- 重金属及其化合物(铅、镉、铬、镍、钴的特定化合物)
- 多环芳烃(PAHs,18 种)
- 芳香胺(由偶氮染料释放)
- 甲醛
- 工业化学品(双酚 A、壬基酚、全氟化合物 PFAS 等)

> 注:具体清单请以 ECHA 官网最新版本为准。**2025 年 1 月 / 6 月** EC 多次新增物质。

### Annex XVII 限制清单(典型条款)

| 物质 | 限制 | 适用类目 |
|---|---|---|
| 偶氮染料(释放芳香胺) | < 30 mg/kg | 服装、纺织品、皮革 |
| 镍释放 | < 0.5 μg/cm²/week(耳钉)/ 0.2(其他) | 金属饰品、珠宝、皮带扣 |
| 镉 | < 100 mg/kg(塑料 / 油漆) | 塑料、油漆、涂层 |
| 铅(珠宝) | < 500 mg/kg(成人)/ < 300 mg/kg(儿童) | 珠宝、饰品 |
| 邻苯二甲酸酯(DEHP/DBP/BBP) | < 0.1% w/w | 塑料、玩具、育儿用品 |
| 多环芳烃(PAHs) | < 1 mg/kg(玩具) | 橡胶、塑料、油墨 |
| 六价铬 | < 3 mg/kg(皮革) | 皮革、鞋类 |
| 富马酸二甲酯(DMF) | < 0.1 mg/kg | 鞋、家具、服装 |
| 甲醛 | < 30 mg/kg(玩具)/ 75 mg/kg(贴身) | 玩具、纺织品 |

> Annex XVII 限制清单**条目众多,具体阈值依子条款而异**,本表仅列典型。本卡不一一罗列。

## 5 步行动框架(总预算 1500-4000 €)

### 第 1 步:产品类目 + 物质风险识别(预算 0 €,1-2 周)

**做什么**:
- 列出所有投放欧盟市场的产品,按**类目风险图谱**归类:
  - **高风险**:服装 / 纺织品、玩具 / 儿童用品、皮革 / 鞋类、金属饰品 / 珠宝、塑料制品
  - **中风险**:电子 / 电器、家具 / 木制品、油漆 / 涂层
  - **低风险**:纯金属 / 纯玻璃 / 纯陶瓷(但也要看工艺)
- **每款产品做"物质构成表"** —— 哪些材料 / 部件可能含 REACH 监管物质
- 字段:SKU / 类目 / 材质 / 上游供应商 / 是否含塑料部件 / 是否含金属部件 / 是否接触皮肤 / 现有 SDS 报告号

**为什么要做这一步**:
- REACH 是"全品类"覆盖,但**类目不同,关键物质不同** —— 不需要把所有 SVHC 都测
- 拆解后可以识别"哪些材料共用" → 跨 SKU 合并检测 → 节省 30-50% 成本
- 拆解过程会发现"哪些部件是高风险"(塑料、皮革、金属涂层、油墨) → 优先做

**避坑**:
- 别只看主产品 —— 包装 / 印刷油墨 / 黏合剂也算 REACH 范围
- 跨 SKU 相似产品 → 单独建行,标注"共用材料"
- 没有"物质清单"概念 → 走完整套检测才发现超支

### 第 2 步:供应链调查 + 物质测试(预算 1000-3000 €,2-4 周)

**必做项**:
- **供应链调查**:发"REACH 物质声明"问卷给上游供应商,要求提供:
  - 物质 SDS(Safety Data Sheet)
  - 物质 / 配制品合规声明
  - 物品的物质构成表(article composition)
- **物质检测**:按类目风险选对应测试
  - 服装:偶氮染料 + 甲醛 + 镍释放
  - 玩具 / 儿童用品:邻苯二甲酸酯 + 镉 + 铅
  - 皮革 / 鞋类:六价铬 + 甲醛 + DMF
  - 金属饰品 / 珠宝:镍释放 + 镉 + 铅
  - 电子:溴化阻燃剂 + 铅 + 镉
  - 家具 / 木制品:甲醛 + VOC

**找谁做**:
- Notified Body:SGS / TÜV SÜD / TÜV Rheinland / Intertek / Bureau Veritas / CTI 华测
- 选有 EU REACH 直接出报告资质的(报告上含 ECHA 认可)
- 报价区间:单 SKU 1000-3000 €,多 SKU 批量谈 600-1500 € / SKU

**关键**:
- 检测报告**保留 10 年**(CE 标志相关文档要求,REACH 配套)
- 报告要**含品牌名 + SKU + 检测日期 + 检测机构印章**
- **跨 SKU 相似材料可以合并检测**(节省 30-50% 成本)
- **供应商的 SDS 不能替代第三方检测**(亚马逊和海关只认 Notified Body 报告)

### 第 3 步:含量计算 + SVHC 通报(预算 0-800 €,1-2 周)

**Article 7 通报触发条件**:
- **单物质 SVHC ≥ 0.1% w/w**(按整个物品计)
- 物质类型:SVHC 候选清单(截至 2025 年 240+ 项)
- 通报对象:ECHA(欧洲化学品管理局)
- 通报时限:**投放市场后 6 个月内**

**通报内容**:
- 物质名称 + CAS 号
- 含量范围(0.1-1%, 1-10%, 等)
- 物品类别
- 安全使用说明
- 制造商 / 进口商信息

**含量计算方法**:
- 按**整个物品重量**算,不是单一均质材料
- 比例公式:某 SVHC 物质在物品中的重量 ÷ 物品总重量 × 100%
- 复杂产品(如儿童玩具)按"最坏情况"算,把所有部件的 SVHC 累加

**通报后义务**:
- **45 天内**回应消费者和下游用户的索取请求
- 网站 / 包装上提供安全使用信息
- 年度更新(物质 / 含量变化时)

**自通报 vs 委托**:
- 简单 1-2 个物质:自通报(ECHA 网站免费提交)
- 复杂多物质 / 多 SKU:委托合规顾问(500-800 € / SKU)
- 模板可参考:ECHA 官网的"Notification of substances in articles"指南

### 第 4 步:Annex XVII 合规审查 + 标签 / 包装(预算 300-1000 €,1-2 周)

**必做项**:
- 核对产品是否含 Annex XVII 限制清单物质
- 物质限量检测(按子条款)
- 标签 / 包装按需更新

**常见标签要求**:
- **镍释放警告**:"本产品含镍,可能引起过敏反应"(持续接触皮肤的金属)
- **邻苯二甲酸酯警告**:儿童用品需声明(若含量 < 0.1% 但 > 0.05% 可选)
- **镉警告**:塑料 / 油漆含镉警告(若有豁免使用)
- **多语种标签**:至少 EN + DE + FR + IT + ES 5 国语言

**印刷方式**:
- **小批量(< 1000 件)**:贴纸 + 手工贴(0.3-0.8 € / 件)
- **中批量(1000-10000 件)**:产品本体直接印刷(0.1-0.4 € / 件)
- **大批量(> 10000 件)**:模具内注塑 / 烫印(0.05-0.2 € / 件)

**避坑**:
- 别只用英语 / 中文 —— **至少 5 国语言**警告
- 印刷前**打样确认**(包装厂提供)
- **Annex XVII 限制清单会持续更新**,需定期复核(ECHA 每季度更新)

### 第 5 步:亚马逊后台 + 详情页同步(预算 0 €,1 周)

**后台字段**(Seller Central):
- `Manage Inventory → Compliance → REACH compliance`
- 每个 SKU 填:类目 / 是否含 SVHC / 是否 Annex XVII 合规 / 物质声明
- 上传:SDS 扫描件 + 检测报告 + ECHA 通报确认(若适用)

**详情页更新**:
- `Compliance` 板块加:REACH 合规说明 + 多语种版本
- 主图区域可加"REACH Compliant"或"EU REACH Certified"标识(亚马逊官方允许)
- 关键文档:SDS PDF + Article 7 通报确认(若适用)

**亚马逊抽查流程**:
- 平台**批量合规团队**按 ASIN + 类目抽查,首次未通过发警告邮件
- 二次未通过 → 限流 / listing 在搜索结果中降权
- 三次未通过 → 强制下架 + 资金冻结 90 天
- 跨境直邮小包 → 海关直接扣货

## 4 个常见误区(基于公开报告)

**误区 1:"我的供应商提供了 SDS,REACH 就合规了"**
错。**SDS 是物质 / 配制品的合规文件,物品(article)需要单独的物质声明 + 检测报告**。你卖的是物品(玩具、衣服、电子产品),不是"化学品",SDS 是不够的。**供应链上"化学品"和"成品"是两套合规要求**。

**误区 2:"我只是贸易商,REACH 是工厂的事"**
错。REACH 下游用户(downstream user)义务**适用于所有在欧盟销售产品的实体**,包括贸易商 / 分销商 / 电商卖家。如果你把产品进口到欧盟 / 在欧盟生产 / 在欧盟销售,你就是**进口商(importer)或分销商(distributor)**,都要承担 REACH 义务。**跨境直邮不算"进口",但亚马逊 FBA 入仓 = 进口**。

**误区 3:"我看了 SVHC 清单,我的产品没有这些物质"**
**清单本身在变**。SVHC 候选清单**每年 1-2 月和 6-7 月**各更新一次,2024-2025 年新增物质包括 PFAS 相关、Pb 相关、邻苯相关等 10+ 项。**2023 年合规 ≠ 2025 年合规**。**建议每 6 个月复核一次 SVHC 清单**。

**误区 4:"REACH 是欧盟的事,英国 / 北美不需要"**
错。**英国有 UK REACH**(2021-01-01 单独运行),**美国有 TSCA**,各有自己的合规要求。本卡只覆盖**欧盟 REACH**。如果你的产品同时卖英国、美国,需要**分别合规**。

## 何时做 / 何时不做

**符合以下任一条件,立刻做**:
- ✅ 你在 Amazon EU 销售实物产品(玩具 / 服装 / 电子 / 家具 / 珠宝 / 任何类目)
- ✅ 你 2026 年后还要继续卖
- ✅ 你的产品含塑料 / 皮革 / 金属 / 油漆 / 油墨等可被 REACH 监管的材料
- ✅ 你重视 listing 长期稳定性

**暂缓**:
- ⚠️ 你 2025-2026 年内准备退出欧洲市场
- ⚠️ 你的产品是纯非实物(数字商品 / 服务)
- ⚠️ 你的产品是食品 / 药品 / 化妆品 / 医疗器械(各有专项法规)

## 如果你只做一件事

**第一步先做"类目风险图谱 + 物质构成表"** —— 把所有产品按类目风险归类,识别"哪些产品需要做哪些物质检测"。这一步**不花钱,1-2 周搞定**,但能让你后续检测预算节省 30-50%,也避免漏检导致全批次扣货。**没有这张表,后面 ECHA 通报和 Annex XVII 审查就是赌博**。

## 行动清单

**30 天内**:
- [ ] 类目清单 + 风险归类(高 / 中 / 低)
- [ ] 物质构成表(每 SKU 5-15 行)
- [ ] 联系 Notified Body 报价(2-3 家)
- [ ] 识别跨 SKU 共用材料
- [ ] 索取上游供应商 SDS

**60 天内**:
- [ ] 下单物质检测(预算 1000-3000 €)
- [ ] 拿到检测报告 + Notified Body 印章
- [ ] 计算 SVHC 含量(按整个物品 w/w)
- [ ] 准备 ECHA 通报(若含量 ≥ 0.1%)

**90 天内**:
- [ ] 提交 ECHA 通报(投放市场后 6 个月内)
- [ ] 标签印制 + 包装更新(预算 300-1000 €)
- [ ] 亚马逊后台字段填好
- [ ] 详情页 Compliance 板块更新
- [ ] 45 天内回应消费者 / 下游用户索取

**120 天内**:全部 SKU 完成 REACH 合规,后台和详情页同步生效
