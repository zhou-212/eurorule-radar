## 一句话结论

**4 个月内必须做 5 件事**:包装清单 + 物质检测 + 空置容积评估 + 标签印制 + 亚马逊后台同步。**总预算 3000-5000 €**,**总时间 6-9 周**。**现在下单 Notified Body,8 月前必须拿报告**。

## 这是啥

EU 2025/40(PPWR,包装与包装废弃物法规)已经在 2025-02-11 生效,**2026-08-12 全面执行**。这条法规替代了原 94/62/EC 包装指令,从"建议性"变成"强制性",覆盖所有投放欧盟市场的带包装商品。

适用你 if:
- ✅ 你在 Amazon EU 5 国(DE/FR/IT/ES/NL)销售带包装的非食品消费品
- ✅ 不论 FBA 还是 FBM,只要包装随商品投放欧盟市场
- ✅ 不论纸盒 / 塑料 / 金属 / 玻璃 / 复合材料,都要符合

不适用 if:
- ❌ 你是 B2B 工业品(部分豁免,看具体类目)
- ❌ 你的产品本身没包装(罕见,绝大多数都有)
- ❌ 你是二手商品且不重新包装(可豁免)

## 为什么跟你有关

**亚马逊公开公告**(Seller Central 2025-09):
> "All products with packaging sold in the EU must comply with PPWR (EU 2025/40) by August 12, 2026. Sellers must provide packaging compliance information through Seller Central."

**没做的后果**:
- 后台 `Packaging compliance` 字段空 → listing 限流
- 物质检测报告缺失 → 亚马逊发"补充资料"邮件
- 持续 60 天不合规 → listing 在欧盟市场不可见
- 欧盟海关抽查 → 罚款 1000-50000 €(看国家)

**4 个月窗口期特别紧的原因**:
- Notified Body 业务量激增,**检测排期 2-4 周**
- 等到 6 月才下单,可能 8 月还没拿到报告
- **建议 4 月底前下单**(写文章日期 2026-07,实际可能 2026 早期)

## 法规核心变化(94/62/EC vs PPWR)

| 维度 | 旧 94/62/EC 指令 | 新 PPWR 法规 | 影响 |
|---|---|---|---|
| 适用 | 各成员国转化立法 | 直接适用全境 27 国 | 统一标准 |
| 物质 | 重金属 ≤ 100 ppm | 重金属 + PFAS + 邻苯二甲酸盐 + SVHC | 检测项增加 1-2 倍 |
| 最小化 | 仅"减少" | 空置容积 ≤ 50%(可量化) | 包装改版 |
| 标签 | 各成员国自定 | 欧盟统一象形标识(2028-08 起) | 未来 2 年要再改 1 次 |
| 回收 | 各国 EPR 缴费 | 回收等级 A-G(2030 起) | 部分产品可能下架 |
| 数字化 | 无 | 回收成分声明(2025 起) | 数据要可查 |

## 5 步行动框架(总预算 3000-5000 €)

### 第 1 步:包装清单 + 分类(预算 0 €,1 周)

**做什么**:
- 列出**所有投放欧盟市场的包装**,按类型分类
- 3 大类:销售包装(零售盒)、成组包装(多件套)、运输包装(快递箱)
- 每类记录:SKU / 包装类型 / 材质 / 重量(g) / 体积(cm³) / 生产商 / 投放国

**工具**:
- 简单 Excel / Google Sheets
- 字段:SKU / 包装类型 / 材质 / 重量 / 体积 / 生产商 / 投放国

**避坑**:
- 别只列"主包装"——亚马逊的"包装"是**所有**投放欧盟市场的包装
- 跨 SKU 共享包装 → 单独建一行,标注"共用"

### 第 2 步:物质检测(预算 2000-4000 €,2-3 周)

**必检项**(基于 PPWR 第 5-7 条 + 附件):
- **重金属**:铅、镉、汞、六价铬 ≤ 100 mg/kg
- **PFAS 全氟烷基物质**:食品接触包装 ≤ 25 ppb(靶向分析)
- **邻苯二甲酸盐**:DEHP / DBP / BBP / DIBP ≤ 0.1%(重量比)
- **REACH 高度关注物质(SVHC)**:≤ 0.1%(按欧盟 ECHA 清单)

**找谁做**:
- Notified Body:SGS / TÜV SÜD / Intertek / Bureau Veritas
- 选有 PPWR 实操经验的(2026 年 1 月起业务量激增,排期长)
- 报价区间:2000-4000 €(首批 5-10 个 SKU 约 3000 €)

**关键**:
- 检测报告**保留 5 年**,亚马逊和欧盟市场监管机构抽查时需要
- 报告要**含品牌名 + SKU + 检测日期 + 检测机构印章**
- 跨 SKU 相似包装可以**合并检测**(节省 30-50% 成本)

### 第 3 步:空置容积评估(预算 0-2000 €,1-2 周)

**PPWR 第 24 条要求**:
```
空置容积 = (V包装 - V产品) / V包装 ≤ 50%
```

**评估方法**:
- 自评:DIN 55401-1 标准(用小颗粒填充法 / 排水法)
- 委托:包装设计公司 / 检测机构(2000-5000 € / SKU)

**优化方案**(按预算从低到高):
1. **换小一号盒子**(50-200 € / SKU)—— 首选
2. **调整产品摆放**(0 €,设计层优化)
3. **真空包装 / 充气包装**(500-2000 € / SKU)—— 适合软商品
4. **改 EPE / 蜂窝纸板替代泡沫**(1000-3000 € / SKU)—— 长期方案

### 第 4 步:标签印制(预算 500-2000 €,1-2 周)

**必印内容**(PPWR 第 12-15 条 + 各国延伸):
- **运营主体**:谁投放市场(你的公司名 + 欧盟地址)
- **批次 / 生产日期**:便于追溯,通常印为"LOT:XXX"或日期
- **包装材质标识**:三角循环符号 + 字母缩写(PET / PP / PE / 纸)
- **2028-08-12 起**:欧盟统一象形标识(暂用各成员国现行 Triman 标志)

**印刷方式**:
- **小批量(< 1000 件)**:贴纸 + 手工贴(0.5-1 € / 件)
- **中批量(1000-10000 件)**:包装盒直接印刷(0.1-0.5 € / 件)
- **大批量(> 10000 件)**:包装盒凹印 / 柔印(0.05-0.2 € / 件)

**避坑**:
- 别只用英语 / 中文——**至少 5 国语言**标签(EN / DE / FR / IT / ES)
- 印刷前**打样确认**(包装厂提供)
- 2028-08-12 前**别全部押注**各国现行标志,留出 18 个月给欧盟统一标识

### 第 5 步:亚马逊后台 + 详情页同步(预算 0 €,1 周)

**后台字段**(Seller Central):
- `Manage Inventory → Compliance → Packaging compliance`
- 每个 SKU 填:包装类型 / 材质 / 是否可回收 / 回收系统注册号
- 法国卖:再填 `Manage your EPR compliance → Packaging`

**详情页更新**:
- `Compliance` 板块加:包装材质 + 回收说明 + 多语种版本
- 主图区域可加"Eco-friendly packaging"标识(亚马逊官方允许的 6 个环保标签之一)

## 4 个常见误区(基于公开报告)

**误区 1:"包装厂说合规就合规"**
错。包装厂说的是"包装本身合规",**不包括你的产品/品牌/欧盟责任人信息**。物质检测报告必须以你的品牌名 + SKU 申请。

**误区 2:"LUCID 注册了 = PPWR 合规"**
错。LUCID 是德国 EPR 包装回收的"缴费证明",PPWR 是"包装本身合规"。**两个不同的法规,两个不同的报告**。

**误区 3:"4 个月够了,不急"**
错。Notified Body 2026 年 1-7 月业务量激增,**检测排期 2-4 周**。等到 6 月才下单,可能 8 月还没拿到报告。

**误区 4:"亚马逊没催就不急"**
亚马逊的合规审核是**分批次的**——你当前没被轮 ≠ 安全。等到被轮,改已经来不及。

## FAQ

### PPWR 跟 EPR 包装(LUCID/Citeo)什么关系?

PPWR 是"包装本身合规"(物质、最小化、标签);EPR 包装是"回收系统缴费"(LUCID/Citeo)。**两个不同法规,都要做**。LUCID 注册了 ≠ PPWR 合规。

### 物质检测找哪家最稳?

SGS / TÜV SÜD / Intertek / Bureau Veritas 都有 PPWR 实操经验。建议**选有欧盟本地实验室**的(德国 TÜV SÜD / 法国 BV),报告欧盟市场认可度更高。

### 4 个月够吗?时间紧怎么压缩?

预算够的话,选**合并检测**(物质 + 空置容积)→ 报告 3-4 周出;**预打样标签**(第 4 步同步做)→ 节省 1 周;**后台字段预先填**(第 5 步提前)→ 8 月前再上检测报告链接。

### PFAS 检测多少钱?

食品接触包装:靶向 PFAS 分析约 500-1000 € / SKU;非食品包装:总 PFAS 筛查约 300-500 € / SKU。**比传统重金属检测贵 2-3 倍**,但 PPWR 强制要求。

### 检测报告能被亚马逊 / 海关抽查时认吗?

Notified Body 出具的报告 + 欧盟本地实验室 + 包含品牌名 / SKU / 检测日期 = 100% 被认。**小作坊或非 Notified Body 的报告,抽查时被拒**。

### 包装上只印英语标签可以吗?

不行。PPWR 要求**投放国本地语言**。如果投放到 DE / FR / IT / ES / NL,标签至少要 EN + DE + FR + IT + ES 5 国语言。**多语种标签成本只增加 5-10%**,别省。
