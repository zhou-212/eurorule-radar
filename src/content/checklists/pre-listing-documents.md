## 适用范围

- **新 SKU 上架 Amazon EU 任一国家**
- **现有 SKU 扩展到新 EU 国家**
- **现有 SKU 包装或设计变更**

## 最后核对日期

**2026-01-15** —— 每条具体执行请按原始来源核对最新版本。

## 来源提示与免责声明

本清单来自 Amazon Seller Central、欧盟委员会公开页。**不构成法律或合规意见**;具体类目(电气、儿童、化妆品、食品)有专项指令,请额外核查。

---

## 1. 基础信息

- [ ] 产品 HS Code / 海关编码(用于跨境物流)
- [ ] 销售国 / 目的国 / 仓储国
- [ ] FBA 入仓计划(若使用 FBA)
- [ ] EAN / UPC / GTIN(亚马逊 GTIN 豁免申请,如适用)

## 2. 平台字段

- [ ] 产品标题(各站点语言)
- [ ] 五点描述(Bullet Points)
- [ ] 产品描述 / A+ Content
- [ ] 关键词 / Search Terms
- [ ] 主图(纯白底 1000x1000px+)+ 辅图
- [ ] 价格(各站点独立,考虑 VAT)
- [ ] 库存数量

## 3. 合规字段(后台上必填)

- [ ] **EU Responsible Person**:姓名 + 邮政地址 + 邮箱(GPSR 强制)
- [ ] **Manufacturer**:制造商信息
- [ ] **Country of Origin**:原产地
- [ ] **年龄警告 / 安全警告**(儿童商品或可被儿童使用)
- [ ] **CE 标志**(适用产品)
- [ ] **能效标签 / EPREL 编号**(电气电子)
- [ ] **EPR 编号**(包装 / 电气 / 电池 / 纺织 / 家具 / 轮胎 / 纸张,按适用类目)
- [ ] **GPSR 安全性信息**(产品安全附件、说明书)

## 4. 产品文件(每 SKU 留存 5-10 年)

- [ ] EU Declaration of Conformity(适用 CE 指令)
- [ ] 测试报告(EN 标准、第三方实验室 ISO 17025)
- [ ] 安全数据表 SDS(化学品 / 化妆品)
- [ ] 食品接触材料声明(FCM)
- [ ] 化妆品产品信息文件 PIF
- [ ] 用户说明书 PDF(多语言)
- [ ] 包装设计稿 / 实物照片
- [ ] 技术文件 Technical File

## 5. 包装与标签

- [ ] Triman / 回收标识(法国)
- [ ] CE 标志(适用产品,≥ 5mm)
- [ ] EU 责任人信息(印刷或贴附)
- [ ] 原产国(英文 + 销售国官方语言)
- [ ] 警告语(销售国语言)
- [ ] 尺寸 / 重量 / 容量单位公制
- [ ] 有效期 / 生产日期(食品 / 化妆品 / 化学品)
- [ ] 批次号 / LOT 编号(可追溯)

## 6. 知识产权

- [ ] 品牌是否在 Amazon Brand Registry
- [ ] 商标 / 专利 / 版权清查
- [ ] 代理品牌授权链
- [ ] 产品图 / 描述 / A+ 是否原创或获授权

## 7. 物流与海关

- [ ] 跨境物流商选型(DHL、UPS、FedEx、顺丰国际、菜鸟等)
- [ ] IOSS / OSS / DDP 模式(按货值、模式确定)
- [ ] 海关申报要素
- [ ] 物流标签(销售国语言,例如德国包装法标识)

## 8. 售后与客服

- [ ] 退换货政策(各 EU 国家不同)
- [ ] 客服联系方式 + 当地官方语言
- [ ] 保修条款
- [ ] 投诉处理流程

---

## 完成后确认

- [ ] 在 Amazon 后台成功保存产品资料
- [ ] 至少 1 个站点成功上架(其它国家可后续启用)
- [ ] 文件备份到公司云盘 / Git 仓库
- [ ] 通知运营 / 客服 / 财务团队
