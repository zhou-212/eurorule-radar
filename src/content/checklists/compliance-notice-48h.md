## 适用范围

- **Amazon Seller Central 通知**(Account Health / Policy Compliance / Listing Deactivation)
- **欧盟监管机构 / 海关 / 商标方投诉**
- **消费者重大投诉**(产品安全、知识产权)
- **平台对产品 / 资料 / 责任人的合规审查**

## 紧急程度

🔴 **所有合规通知都需要在 48 小时内做出反应,即使只是提交初步回应**(否则平台可能升级处置)。

## 最后核对日期

**2026-01-15** —— 平台具体处理流程以 Amazon Seller Central 公告为准。

本清单基于 Amazon 公开 Account Health 文档整理。**不构成法律意见**;严重案件(产品安全召回、海关扣留、商标诉讼)请立即咨询专业律师。

## ⏰ 0-2 小时:识别通知类型

- [ ] 通知类型标记:
  - [ ] **Listing 警告 / Listing 暂停** —— 多数可申诉
  - [ ] **账号健康警告(Account Health)** —— 优先处理
  - [ ] **政策违规通知** —— 多数含具体违规条款
  - [ ] **产品安全投诉** —— 严重,48h 内必须回应
  - [ ] **品牌 / 知识产权侵权** —— 需法律评估
  - [ ] **海关 / 监管机构** —— 立即升级到合规负责人 / 律师
- [ ] 把通知转发到:合规负责人 + 运营负责人 + 法务(如有)
- [ ] 截图原文 + 保存邮件原文(后续申诉要用)
- [ ] 在内部协作工具创建事件(标题/时间线/负责人)

## ⏰ 2-6 小时:初步评估

- [ ] 通知要求的**动作类型**:
  - [ ] 提交文件(CE / 测试报告 / DoC / 责任人证明)
  - [ ] 修改 listing(标题 / 图片 / 描述 / 关键词)
  - [ ] 删除 listing
  - [ ] 召回 / 销毁库存
- [ ] **评估证据**:
  - [ ] 能否提供平台要求的文件?
  - [ ] listing 描述是否真有不准确?
  - [ ] 产品是否真存在安全问题?
  - [ ] 是否有平台错误的可能(例如:同一 ASIN 多店铺被错告)?
- [ ] 评估**业务影响**:受影响 listing 数 / 月销量 / 利润 / 品牌风险

## ⏰ 6-24 小时:决定应对方案

- [ ] **方案 A:申诉 / 提交文件**
  - [ ] 列出平台要求的文件清单
  - [ ] 准备 Plan of Action(POA)模板:
    - 根本原因(Root Cause)
    - 纠正措施(Corrective Action)
    - 预防措施(Preventive Action)
  - [ ] 提交申诉(注意:Amazon 政策通常只允许一次主要申诉,**申诉前必须充分准备**)
- [ ] **方案 B:承认违规,删除 listing**
  - [ ] 立即删除问题 listing
  - [ ] 同步内部库存(停止发货)
  - [ ] 必要时召回 FBA 库存(联系 Amazon 移除)
- [ ] **方案 C:法律评估(严重案件)**
  - [ ] 联系欧盟律师 / 知识产权律师
  - [ ] 准备正式回应函
  - [ ] 评估是否提起抗辩

## ⏰ 24-48 小时:提交回应 + 内部复盘

- [ ] 提交申诉 / 计划书(留底)
- [ ] 跟踪回复(平台通常 2-5 个工作日,严重 7-14 天)
- [ ] **内部复盘**:
  - [ ] 这类问题为什么会发生?
  - [ ] 流程哪里有漏洞?
  - [ ] 如何避免下次再犯?
- [ ] 更新内部 SOP / 培训
- [ ] 如果是账号健康问题,跟踪 ODR(订单缺陷率)趋势

## ⏰ 48 小时后:持续监控

- [ ] 每天检查 Amazon 通知(直到案件关闭)
- [ ] 准备"如果被拒"的下一步方案
- [ ] 必要时考虑 Amazon 卖家申诉合作伙伴(Appeal Service)
- [ ] 涉及产品安全的:评估是否需要公开召回 / 通知消费者

## 常见误区

- ❌ "先不管,等 Amazon 自动恢复" —— 多数情况会升级到 Listing 暂停
- ❌ "找服务商花钱搞定" —— Amazon 申诉多数要求**真实合规文件 + 真实整改**
- ❌ "删了 listing 就完事" —— 账号健康(OZONE)会持续累积,影响整店
- ❌ "只回一封邮件" —— 平台希望看到**根因 + 纠正 + 预防**三段式
